Navigating Panning can be done by clicking the MMB and dragging. Zooming can be done by scrolling Wheel up or down. Also, as in the 3D View, you can use NumpadPlus or NumpadMinus to zoom.
Gizmos Next to the Sidebar region at the top, there are gizmos that allow panning and zooming more comfortably when e.g. no mouse wheel is available.

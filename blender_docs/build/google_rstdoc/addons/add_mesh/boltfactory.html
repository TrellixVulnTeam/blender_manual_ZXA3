Bolt Factory
Introduction This add-on creates bolts and nuts with options for bolt/nut, bit type and head type.
Activation Open Blender and go to Preferences then the Add-ons tab. Click Add Mesh then BoltFactory to enable the script.
Interface Located in the 3D View ‣ Add ‣ Mesh menu.

Archimesh
Introduction This tool is specially designed to generate architecture elements, like: Rooms Doors Windows Kitchen cabinets Shelves Columns stairs Tile roofs Books Lamps Venetian blinds Roller curtain Japanese curtains The original video documentation can be found here: Video Playlist . Note that the videos were created for Blender 2.7 series but are still a valid resource.
Activation Open Blender and go to Preferences then the Add-ons tab. Click Add Mesh then Archimesh to enable the script.
Interface Located in the 3D View ‣ Sidebar ‣ Create tab. Reference Category Add Mesh Description Generate rooms, doors, windows and architecture objects.

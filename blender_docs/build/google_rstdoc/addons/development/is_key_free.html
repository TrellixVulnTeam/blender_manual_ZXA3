Is Key Free
Introduction Primarily used to check the availability of hotkeys and the editor types they are associated with.
Activation Open Blender and go to Preferences then the Add-ons tab. Click Development then Is key Free to enable the script.
Description Search All the registered keys associated with the pattern entered in the Key field and selected modifier shortcuts will be displayed. Quick Access Alternatively, a list with available shortcuts is available for quick access of keys from the Quick Access selector. Reference Category Development Description Find free shortcuts, inform about used and print a key list.

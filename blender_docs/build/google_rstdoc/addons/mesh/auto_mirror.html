Activation Open Blender and go to Preferences then the Add-ons tab. Click Mesh then Auto Mirror to enable the script.
Usage In Object Mode, select the object you want to mirror. Choose settings from the Auto Mirror panel in the Sidebar ‣ Edit tab . Press Auto Mirror and depending on the settings, your object will be in Edit Mode, with mirror set up for you. Reference Category Mesh Description Super fast cutting and mirroring for mesh.

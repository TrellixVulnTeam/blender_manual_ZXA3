Skinify Rig
Introduction This add-on creates a mesh object “Skinify Guy” covering an armature with a mesh. It works with Rigify: It works with BVH imports: It works with manuelbastionilab add-on:
Activation Open Blender and go to Preferences then the Add-ons tab. Click Object then Scatter Objects to enable the script.
Usage Select your armature in Pose Mode and skin the selected or all bones with a per-bone mesh or whole cover. If bones in your rig give unwanted results, often scaling and re-positioning the bones in Edit Mode can help. If your doing this to create your own “Skinify Guy” you can also delete bones. It will work with custom shapes but results may vary. More to come… Reference Category Object Description Creates a mesh object from selected bones.

Primitives Reference Mode Object Mode and Edit Mode Panel Toolbar ‣ Create ‣ Add Primitive/Metaball Menu Add ‣ Meta Hotkey Shift-A There are five predefined meta “primitives” (or configurations) available in the Add ‣ Meta submenu:
Options Primitive Meta Ball Adds a meta with a point underlying structure. Meta Capsule Adds a meta with a line segment underlying structure. Meta Plane Adds a meta with a planar underlying structure. Meta Ellipsoid Adds a meta with an ellipsoidal underlying structure. Meta Cube Adds a meta with a volumetric cubic underlying structure. A more detailed explanation of each primitive in the Structure page. Radius, Align to View, Location, Rotation See Common Object Options .

Make Edge/Face Reference Mode Edit Mode Menu Vertex â€£ New Edge/Face from Vertices Hotkey F This is a context-sensitive tool which creates geometry by filling in the selection. When only two vertices are selected it will create an edge, otherwise it will create faces. The typical use case is to select vertices and press F , however, Blender also supports creating faces from different selections to help quickly build up geometry.

Dissolve Existing Faces When you have a region of existing faces, creating a face on this selection will remove the shared vertices and edges, creating a single face. This is simply a convenience for accessing Dissolve Faces .

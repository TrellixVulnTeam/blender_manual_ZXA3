Clip Strips Clip can be modified within the Movie Clip Editor .
Options This strip has no options.
Mask Strips The Mask strip generates a mask image from the selected mask data-block generated in the Movie Clip Editor . This works similar to the Mask Node but without the options available for finer control. The mask image is always generated at the render resolution, scaling along with different proxy levels.
Options Mask Data-block menu to select a mask.

Object Use the color that can be set per object in the Viewport Display Object panel. Material Use the color that can be set per material in the Viewport Display Material panel. Random A random color will be selected for every object in the scene. Vertex Display the active vertex colors of an object. When an object has no active vertex colors it will be rendered in the color that is set in the Viewport Display Material panel. Texture Display the active texture. When an object has no active texture the object will be rendered with the settings in the Viewport Display Material panel. The Texture option is not available when Lighting is set to MatCap .

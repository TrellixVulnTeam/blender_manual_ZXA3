Color Balance Node The Color Balance node can adjust the color and values of an image.
Inputs Factor Controls the amount of influence the node exerts on the output image. Color Standard image input.
Properties Two different correction formulas could be selected. Lift/Gamma/Gain Lift Increases the value of dark colors. Gamma Will adjust midtones. Gain Adjusts highlights. Offset/Power/Slope (ASC-CDL) Offset Summand. (Adjusts the overall brightness.) Basis Additional offset, allows to specify a negative Offset value. Power Over-all exponent. (Adjusts the midtones.) Slope Multiplier. (Adjusts the highlights.)
Outputs Color Standard output image.

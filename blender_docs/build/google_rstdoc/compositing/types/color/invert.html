Invert Node The Invert Node inverts the colors in the input image, producing a negative.
Inputs Factor Controls the amount of influence the node exerts on the output image. Color Standard image input.
Properties In the compositing context this node has the following properties. RGB De/activation of the color channel inversion. Alpha De/activation of the alpha channel inversion.
Outputs Color Standard image output.

Dilate/Erode Node The Dilate/Erode node provides a morphology (mathematical shape analysis) filter.
Inputs Mask Single color channel (or a black-and-white image) input.
Properties Mode Step, Threshold, Distance, Feather Distance The Distance is the filter radius. A positive value of Distance dilates (expands) the influence of a pixel on its surrounding pixels. A negative value erodes (shrinks) its influence. Edge Edge to inset. .. TODO2.8 Explain. Falloff Falloff type the feather. .. TODO2.8 Explain.

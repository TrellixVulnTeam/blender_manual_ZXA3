Common Options Some options shared by many fluid types.
Volume Initialization Type A common option among the different fluid types is Volume Initialization . Volume The inside of the object is initialized as fluid all. This works only if the mesh is closed. Shell It is initialized as a thin fluid layer of the surface of the mesh. This can also be used if the mesh is open. Both It is a state, such as the sum of the Volume and Shell. This also must be a closed mesh.

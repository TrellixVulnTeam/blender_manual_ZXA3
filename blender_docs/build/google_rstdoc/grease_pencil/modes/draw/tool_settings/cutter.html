Cutter Tool Reference Mode Draw Mode Panel Toolbar ‣ Tools ‣ Draw Tools ‣ Cutter The Cutter tool delete points in between intersecting strokes.
Usage Draw a dotted line around the strokes you want to be cut. After releasing the mouse/pen all the points on the selected strokes will be deleted until another intersecting stroke is found.

Draw Brush Draw brushes are the special type of brushes that uses Grease Pencil for drawing tools. The brush can be changed in the Tool Settings. The different draw brushes (pencil, Ink, marker, etc.) are settings variations of the same Draw Brush . you can create many brushes, each with unique settings to obtain different artistic result while drawing.

Input Samples Controls how often the input device is read to generate points on the stroke. Higher values give a higher precision (more points) but produce an irregular stroke, while lower values give a lower precision (fewer points) but produce a soften stroke. (0 disabled extra input device samples.) You have to set up this value according to your input device to obtain the right balance between accuracy and softness for your strokes.

Blur Visual Effect The Blur Visual Effect applies a Gaussian blur to the object.
Options Factor Control the blur scale in pixels on the X and Z axis. X, Z Samples Number of Blur samples (0 disabled the blur effect). Lock Focal Plane When enabled, the blur effect uses the focal plane distance of the actual camera to simulate a depth of field effect. Only available in camera view.

Swirl Visual Effect The Swirl Visual Effect applies a swirling pattern to the object. The effect use an object as the center of the swirl.
Options Object Sets the object to use as the center of the swirl. Radius External radius size of the swirl. Angle Rotation angle of the swirl. A value of 0 shows no swirl. Transparent When enabled, the Object is fully transparent outside the radius of the swirl.

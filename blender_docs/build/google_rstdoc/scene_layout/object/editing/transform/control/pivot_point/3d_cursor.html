3D Cursor Reference Mode Object Mode and Edit Mode Header Pivot Point ‣ 3D Cursor Hotkey Period The 3D cursor is the most intuitive of the pivot points. With the 3D cursor selected as the active pivot point (from either the Editor’s Header or via Period ), simply position the 3D cursor and then do the required transformation. All rotation and scaling transformations will now be done relative to the location of the 3D cursor.

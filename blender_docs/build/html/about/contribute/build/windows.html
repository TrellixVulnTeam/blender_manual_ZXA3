Xây Dựng trên Windows -- Building on Windows¶

Chuyển đổi các tập tin RST sang các trang HTML đẹp đẽ. Có hai cách để thi hành quá trình xây dựng.

Trình Duyệt Tập Tin -- File browser

Thi hành bản make.bat trong thư mục C:\blender_docs.

Mẹo

Tạo một đường tắt trên mặt bàn làm việc tới lệnh tạo -- make.

Cửa Sổ Bàn Giao Tiếp -- Command prompt

  1. Hoặc mở một cửa sổ dòng lệnh và chuyển vào thư mục của kho với lệnh cd C:\blender_docs.

  2. Xây dựng bằng cách sử dụng lệnh sau:

    make

Quá trình xây dựng lần đầu tiên sẽ kéo dài vài phút (hoặc sau những thay đổi lớn), song lần xây dựng sau sẽ chỉ kéo dài vài giây mà thôi.

Ghi chú

Nếu bạn gặp phải một báo cáo lỗi nào đó có câu kết thúc là Lỗi về Kiểu Dữ Liệu: đòi hỏi một số nguyên (integer), song lại nhận được kiểu chuỗi văn bản (str) -- TypeError: an integer is required (got type str), bạn có thể cần phải cài đặt phiên bản cũ của Babel (Thư Viện Quốc Tế Hóa của Python (Python Internationalization Library)).

Để là điều này, hãy đơn thuần thi hành:

pip install sphinx "babel<2.0"

Duyệt Thảo Bản Hướng Dẫn Sử Dụng Cục Bộ -- Viewing the Local Manual¶

Một khi các tài liệu đã được xây dựng, toàn bộ các tập tin HTML sẽ nằm trong C:\blender_docs\build\html. Hãy thử mở \build\html\index.html trong trình duyệt mạng của bạn và đọc bản hướng dẫn sử dụng.

Tiếp tục với bước tới: Biên Soạn -- Editing.

Giới Thiệu -- Introduction Mesh Modeling typically begins with a Mesh Primitive shape (e.g. circle, cube, cylinder...). From there you might begin editing to create a larger, more complex shape.
-- Modeling Modes The 3D View has three principal modes that allow for the creation, editing and manipulation of the mesh models. Each of the three modes have a variety of tools. Some tools may be found in one or more of the modes. Modes that used for modeling: Chế Độ Vật Thể -- Object Mode Supports basic operations such as object creation, joining objects, managing shape keys, UV/color layers. Chế Độ Biên Soạn -- Edit Mode Used for the majority of mesh editing operations. Chế Độ Điêu Khắc -- Sculpt Mode Instead of dealing with individual mesh elements, support sculpting with brushes (not covered in this chapter) .

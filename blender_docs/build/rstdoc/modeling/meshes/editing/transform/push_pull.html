Đẩn/Kéo -- Push/Pull Tham Chiếu -- Reference Chế Độ -- Mode Chế Độ Vật Thể và Biên Soạn -- Object and Edit Modes Bảng -- Panel Toolbar ‣ Tools ‣ Transform ‣ Push/Pull Trình Đơn -- Menu Vật Thể/Khung Lưới (Object/Mesh) ‣ Biến Hóa (Transform) ‣ Kéo (Push Pull) Push/Pull will move the selected elements (objects, vertices, edges or faces) closer together (Push) or further apart (Pull). Specifically, each element is moved towards or away from the center by the same distance. This distance is controlled by moving the mouse up (Push) or down (Pull), numeric input or through slider control.

Điêu Khắc UV -- UV Sculpting Tham Chiếu -- Reference Chế Độ -- Mode Paint Mode and Mask Mode Bảng -- Panel Giá Công Cụ (Tools Shelf) ‣ Tùy Chọn (Options) ‣ Điêu Khắc UV (UV Sculpt) , Giá Công Cụ (Tools Shelf) ‣ Công Cụ (Tools) Trình Đơn -- Menu UV (UVs) ‣ Điêu Khắc UV (UV Sculpt) Phím Tắt -- Hotkey Q The UV Sculpting "mode" allow you to grab, pinch and smooth UVs, just like Sculpt Mode. It can be activated with Q or by checking UV Sculpt in the UVs menu.
Điêu Khắc UV -- UV Sculpt When UV sculpting is activated, the Toolbar shows the brush tool selection and options. Khóa Ranh Giới -- Lock Borders Locks the boundary of UV islands from being affected by the brush. This is very useful to preserve the shape of UV islands. Điêu Khắc Toàn Bộ các Hải Đảo -- Sculpt All Islands To edit all islands and not only the island nearest to the brush center when the sculpt stroke was started. Công Cụ Điêu Khắc UV -- UV Sculpt Tools Brushes that operate on UVs. All brushes use the Airbrush Stroke Method: they continue to act as long as you keep LMB pressed. Nắm Giữ -- Grab G The Grab brush moves UVs around. Nới Lỏng -- Relax S , Shift-LMB The Relax brush makes UVs more evenly distributed. The algorithm relies on space, not stretch minimization, so most probably a minimize stretch will have to be run for optimal results. However it is great to use after stitching islands, or when unwrap produces cluttered results to smooth the distribution of UVs. Phương Pháp Hồi Phục -- Relaxation Method There are two relax algorithms: Laplacian, HC Véo -- Pinch P The Pinch brush moves UVs toward the brush's center. The pinch brush can be inverted by pressing Ctrl-LMB . Hiển Thị Bút Vẽ -- Show Brush Ẩn giấu con trỏ điêu khắc.

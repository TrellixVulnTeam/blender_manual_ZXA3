Dọn Dẹp -- Clean up These tools are to help cleanup degenerate geometry and fill in missing areas of a mesh.
Tiêu Hao Hình Học -- Decimate Geometry Tham Chiếu -- Reference Chế Độ -- Mode Chế Độ Biên Soạn -- Edit Mode Trình Đơn -- Menu Khung Lưới (Mesh) ‣ Dọn Dẹp (Clean up) ‣ Tiêu Hao Hình Học (Decimate Geometry) The Decimate Geometry tool allows you to reduce the vertex/face count of a mesh with minimal shape changes. Tỷ Số -- Ratio Ratio of triangles to reduce to. Nhóm Điểm Đỉnh -- Vertex Group Use the active vertex group as an influence. Trọng Lượng -- Weight Strength of the vertex group. Đảo Nghịch -- Invert Inverts the vertex group. Đối Xứng -- Symmetry Maintain symmetry on either the X , Y , or Z axis. This tool works similar to the Decimate Modifier .
Lấp Lỗ -- Fill Holes Tham Chiếu -- Reference Chế Độ -- Mode Chế Độ Biên Soạn -- Edit Mode Trình Đơn -- Menu Khung Lưới (Mesh) ‣ Dọn Dẹp (Clean up) ‣ Lấp Lỗ (Fill Holes) This tool can take a large selection and detect the holes in the mesh, filling them in. This is different from the face creation operator in three important respects: Holes are detected, so there is no need to manually find and select the edges around the holes. Holes can have a limit for the number of sides (so only quads or tris are filled in for example). Mesh data is copied from surrounding geometry (UVs, vertex colors, multi-res, all layers), since manually creating this data is very time-consuming.
Tạo Bề Mặt Bằng Phẳng -- Make Planar Faces Tham Chiếu -- Reference Chế Độ -- Mode Chế Độ Biên Soạn -- Edit Mode Trình Đơn -- Menu Khung Lưới (Mesh) ‣ Dọn Dẹp (Clean up) ‣ Tạo Bề Mặt Bằng Phẳng (Make Planar Faces) The Make Planar Faces iteratively flattens faces. This can happen with faces over three vertices and it is a common convention that faces should be kept planar. Hệ Số -- Factor Distance to move the vertices each iteration. Số Lần Lặp Lại -- Iterations Number of times to repeat the operation.
Phân Chia các Bề Mặt Không Phẳng -- Split Non-Planar Faces Tham Chiếu -- Reference Chế Độ -- Mode Chế Độ Biên Soạn -- Edit Mode Trình Đơn -- Menu Khung Lưới (Mesh) ‣ Dọn Dẹp (Clean up) ‣ Phân Chia các Bề Mặt Không Phẳng (Split Non-Planar Faces) This tool avoids ambiguous areas of geometry by splitting non-flat faces when they are bent beyond a given limit.
Phân Chia các Bề Mặt Lõm -- Split Concave Faces Tham Chiếu -- Reference Chế Độ -- Mode Chế Độ Biên Soạn -- Edit Mode Trình Đơn -- Menu Khung Lưới (Mesh) ‣ Dọn Dẹp (Clean up) ‣ Phân Chia các Bề Mặt Lõm (Split Concave Faces) This tool can be used to convert any concave face to convex by splitting the concave into two or more convex faces.
-- Delete Loose Geometry Tham Chiếu -- Reference Chế Độ -- Mode Chế Độ Biên Soạn -- Edit Mode Trình Đơn -- Menu Khung Lưới (Mesh) ‣ Dọn Dẹp (Clean up) ‣ Xóa Rời Rạc (Delete Loose) This tool removes disconnected vertices and edges (optionally faces).
Hòa Tan Suy Biến -- Degenerate Dissolve Tham Chiếu -- Reference Chế Độ -- Mode Chế Độ Biên Soạn -- Edit Mode Trình Đơn -- Menu Khung Lưới (Mesh) ‣ Dọn Dẹp (Clean up) ‣ Hòa Tan Suy Biến (Degenerate Dissolve) This tool collapses / removes geometry which you typically will not want. Edges with no length. Faces with no areas (faces on a point or thin faces). Face corners with no area.

Hình Cơ Bản -- Primitives Tham Chiếu -- Reference Chế Độ -- Mode Object Mode and Edit Mode Bảng -- Panel Toolbar ‣ Create ‣ Add Primitive/Metaball Trình Đơn -- Menu Cộng Thêm (Add) ‣ Siêu Vật Thể (Meta) Phím Tắt -- Hotkey Shift-A There are five predefined meta "primitives" (or configurations) available in the Cộng Thêm (Add) ‣ Siêu Vật Thể (Meta) submenu:
Tùy Chọn -- Options Hình Cơ Bản -- Primitive -- Meta Ball Adds a meta with a point underlying structure. Meta Capsule Adds a meta with a line segment underlying structure. -- Meta Plane Adds a meta with a planar underlying structure. -- Meta Ellipsoid Adds a meta with an ellipsoidal underlying structure. -- Meta Cube Adds a meta with a volumetric cubic underlying structure.

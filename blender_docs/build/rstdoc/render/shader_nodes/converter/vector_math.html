Nút Toán Vectơ -- Vector Math Node The Vector Math node performs the selected math operation on the input vectors.
Đầu Vào -- Inputs The inputs of the node are dynamic. Some inputs are only available in certain operations. For instance, the Scale input is only available in the Scale operator. Vectơ -- Vector Input vector A. Vectơ -- Vector Input vector B. Scale Input Scale.
Tính Chất -- Properties Thao Tác -- Operation The vector math operator to be applied on the input vectors. Cộng Thêm -- Add The sum of A and B. Trừ Khấu -- Subtract The difference between A and B. Multiply The entrywise product of A and B. (A.x * B.x, A.y * B.y, A.z * B.z) Divide The entrywise division of A by B. Division by zero returns zero. (A.x / B.x, A.y / B.y, A.z / B.z) Tích có Hướng -- Cross Product The cross product of A and B. Project The projection of A onto B. Reflect The reflection of A around the normal B. B need not be normalized. Tích Vô Hướng -- Dot Product The dot product of A and B. Distance The distance between A and B. Length The length of A. Scale The result of multiplying A by the scalar input Scale . Bình Thường Hóa -- Normalize The result of normalizing A. Snap The result of rounding A to the largest integer multiple of B less than or equal A. Floor The entrywise floor of A. Ceil The entrywise ceiling of A. Modulo The entrywise modulo of A by B. Fraction The fractional part of A. Absolute The entrywise absolute value of A. Minimum The entrywise minimum from A and B. Maximum The entrywise maximum from A and B.
Đầu Ra -- Outputs The output of the node is dynamic. It is either a vector or a scalar depending on the operator. For instance, the Length operator have a scalar output while the Add operator have a vector output. Vectơ -- Vector Output vector. Giá Trị -- Value Output value.

Nút Bước Sóng -- Wavelength Node Cycles Only The Wavelength node converts a wavelength value to an RGB value. This can be used to achieve a specific color on the light spectrum.
Đầu Vào -- Inputs Bước Sóng -- Wavelength The color wavelength from 380 to 780 nanometers.
Tính Chất -- Properties Nút này không có tính chất nào cả.
Đầu Ra -- Outputs Màu Sắc -- Color RGB color output.

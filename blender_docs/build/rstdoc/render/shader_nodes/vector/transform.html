Nút Biến Hóa Vectơ -- Vector Transform Node The Vector Transform node allows converting a Vector, Point, or Normal between World and Camera and Object coordinate space.
Đầu Vào -- Inputs -- Vector Input Đầu vào vectơ tiêu chuẩn.
Tính Chất -- Properties Thể Loại -- Type Specifies the input/output type. Vector, Point, Normal. Chuyển Đổi Từ -- Convert From Coordinate Space to convert from: World, Object, Camera. Chuyển Đổi Sang -- Convert To Coordinate Space to convert to: World, Object, Camera.
Đầu Ra -- Outputs -- Vector Output The transformed output vector.
Ví Dụ -- Examples Nội dung cần viết thêm <2.8.

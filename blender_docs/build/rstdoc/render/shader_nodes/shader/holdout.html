Giữ Chỗ Trống -- Holdout The Holdout shader node is used to create a "hole" in the image with zero alpha transparency, which is useful for compositing (see alpha channel ). Note that the holdout shader can only create alpha when Tính Chất (Properties) ‣ Kết Xuất (Render) ‣ Phim (Film) ‣ Trong Suốt (Transparent) is enabled. If it is disabled, the holdout shader will be black.

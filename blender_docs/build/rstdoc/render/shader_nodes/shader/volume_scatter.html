Tán Xạ Thể Tích -- Volume Scatter The Volume Scatter node allows light to be scattered as it passes through the volume. Typical usage would be to add fog to a scene. It can also be used with the Volume Absorption node to create smoke.
Đầu Vào -- Inputs Màu Sắc -- Color Color of the volume. Tỷ Trọng -- Density The density of the scatter effect. Dị Hướng -- Anisotropy Controls the look of the scatter effect depending on the direction of the light passing through it.
Tính Chất -- Properties Thể Tích -- Volume The Volume Shader output must be plugged into the Volume Input of the Material or World output node.

Nền -- Background The Background shader node is used to add background light emission. This node should only be used for the world surface output.
Đầu Vào -- Inputs Màu Sắc -- Color Color of the emitted light. Sức Mạnh/Cường Độ -- Strength Strength of the emitted light.
Tính Chất -- Properties Nút này không có tính chất nào cả.
Đầu Ra -- Outputs Nền -- Background Đầu ra chuẩn của bộ tô bóng.

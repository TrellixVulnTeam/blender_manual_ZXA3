Nút Khung Dây -- Wireframe Node The Wireframe node is used to retrieve the edges of an object as it appears to Cycles. As meshes are triangulated before being processed by Cycles, topology will always appear triangulated when viewed with the Wireframe node .
Đầu Vào -- Inputs Nút này không có đầu vào.
Tính Chất -- Properties Kích Thước Điểm Ảnh -- Pixel Size When enabled, the size of edge lines is set in screen space. Kích Thước -- Size Thickness of the edge lines.
Đầu Ra -- Outputs Hệ Số -- Factor Black-and-white mask showing white lines representing edges according to the object's topology .
Ví Dụ -- Examples Nội dung cần viết thêm <2.8.

-- Camera Data Node The Camera Data node is used to get information about the position of the object relative to the camera. This could be used for example to change the shading of objects further away from the camera, or make custom fog effects.
Đầu Vào -- Inputs Nút này không có đầu vào.
Tính Chất -- Properties Nút này không có tính chất nào cả.
Đầu Ra -- Outputs Vectơ từ Điểm Nhìn -- View Vector A Camera space vector from the camera to the shading point. Độ Sâu Z từ Điểm Nhìn -- View Z Depth The distance each pixel is away from the camera. Khoảng Cách từ Điểm Nhìn -- View Distance Distance from the camera to the shading point.

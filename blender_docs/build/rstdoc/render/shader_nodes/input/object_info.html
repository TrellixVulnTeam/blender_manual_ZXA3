Nút Thông Tin Vật Thể -- Object Info Node The Object Info node gives information about the object instance. This can be useful to give some variation to a single material assigned to multiple instances, either manually controlled through the object index, based on the object location, or randomized for each instance. For example a Noise texture can give random colors or a Color Ramp can give a range of colors to be randomly picked from.
Đầu Vào -- Inputs Nút này không có đầu vào.
Tính Chất -- Properties Nút này không có tính chất nào cả.
Đầu Ra -- Outputs Vị Trí -- Location Location of the object in world space. Color Object color, same as Color in the Properties Editor ‣ Object ‣ Viewport Display . Chỉ Số Vật Thể -- Object Index Object pass index, same as Pass Index in the Trình Biên Soạn Tính Chất (Properties Editor) ‣ Vật Thể (Object) ‣ Mối Quan Hệ (Relations) . Chỉ Số Nguyên Liệu -- Material Index Material pass index, same as Pass Index in the Trình Biên Soạn Tính Chất (Properties Editor) ‣ Nguyên Liệu (Material) ‣ Sắp Đặt (Settings) . Ngẫu Nhiên -- Random Random number unique to a single object instance. Note that this node only works for material shading nodes; it does nothing for light and world shading nodes.

Nút Tiếp Tuyến -- Tangent Node The Tangent node generates a tangent direction for the Anisotropic BSDF.
Đầu Vào -- Inputs Nút này không có đầu vào.
Tính Chất -- Properties -- Direction Type The tangent direction can be derived from a cylindrical projection around the X, Y, or Z axis (radial), or from a manually created UV Map for full control.
Đầu Ra -- Outputs Tiếp Tuyến -- Tangent The tangent direction vector.

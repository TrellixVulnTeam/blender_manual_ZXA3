Nút Ánh Xạ UV -- UV Map Node The UV Map node is used to retrieve specific UV maps. Unlike the Texture Coordinate Node which only provides the active UV map, this node can retrieve any UV map belonging to the object using the material.
Đầu Vào -- Inputs Nút này không có đầu vào.
Tính Chất -- Properties From Instancer Cycles Only See the From Instancer option of the Texture Coordinate Node . Ánh Xạ UV -- UV Map UV map to use.
Đầu Ra -- Outputs UV UV mapping coordinates from the specified UV map.

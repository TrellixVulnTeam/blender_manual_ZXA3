-- Magic Texture Node The Magic Texture node is used to add a psychedelic color texture.
Đầu Vào -- Inputs Vectơ -- Vector Tọa độ chất liệu để lấy mẫu vật chất liệu tại đó; Trong trường hợp ổ cắm không có kết nối nào thì sử dụng tọa độ của chất liệu sinh tạo (Generated). Tỷ Lệ -- Scale Scale of the texture. Sự Bóp Méo -- Distortion Amount of distortion.
Tính Chất -- Properties Chiều/Độ Sâu -- Depth Number of iterations.
Đầu Ra -- Outputs Màu Sắc -- Color Đầu ra của màu chất liệu. Hệ Số -- Factor Cường độ xuất của chất liệu.

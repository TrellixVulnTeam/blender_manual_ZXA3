Nút Chất Liệu Gạch -- Brick Texture Node The Brick Texture is used to add a procedural texture producing bricks.
Đầu Vào -- Inputs -- Color 1/2 and Mortar Color of the bricks and mortar. Tỷ Lệ -- Scale Tỷ Lệ Vật Thể Tổng Quan Kích Thước Vữa -- Mortar Size The size of the filling between the bricks known as "mortar"; 0 means no mortar. Vữa Mịn -- Mortar Smooth Blurs/softens the edge between the mortar and the bricks. This can be useful with a texture and displacement textures. Thiên Lệch -- Bias The color variation between Color 1/2 . Values of -1 and 1 only use one of the two colors; values in between mix the colors. Chiều Rộng của Gạch -- Brick Width The width of the bricks. Chiều Cao của Hàng -- Row Height The height of the brick rows.
Tính Chất -- Properties Dịch Chuyển -- Offset Determines the brick offset of the various rows. Tần Số -- Frequency Determines the offset frequency. A value of 2 gives an even/uneven pattern of rows. Nén ép -- Squash Amount of brick squashing. Tần Số -- Frequency Brick squashing frequency.
Đầu Ra -- Outputs Màu Sắc -- Color Đầu ra của màu chất liệu. Hệ Số -- Factor Mortar mask (1 = mortar).

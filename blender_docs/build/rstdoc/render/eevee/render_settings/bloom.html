Sáng Lóa -- Bloom Bloom is a post-process effect that diffuses very bright pixels. This mimics lens artifacts of real cameras. This allows a better sense of what the actual intensities of the pixels are. Tham Chiếu -- Reference Bảng -- Panel Kết Xuất (Render) ‣ Sáng Lóa (Bloom) Giới Hạn -- Threshold Filters out pixels under this level of brightness. -- Knee Makes transition between under/over-threshold gradual. Bán Kính -- Radius Bloom spread distance. Màu Sắc -- Color Color applied to the bloom effect. Cường Độ -- Intensity Blend factor. Hạn Định -- Clamp Maximum intensity a bloom pixel can have.

Máy Chơi Hoạt Họa -- Animation Player The Topbar ‣ Render ‣ View Animation menu will play back the rendered animation in a new window. You can also drop images or movie files in a running animation player. It will then restart the player with the new data. An external player can also be used instead of the one included in Blender. To do this, select it in the Preferences .

Siêu Dữ Liệu -- Metadata The Metadata panel includes options for writing metadata into render output. Only some image formats support metadata: See image formats . Siêu Dữ Liệu Sử Dụng -- Enabled Metadata Stamping can include the following data: Ngày -- Date Includes the current date and time. Thời Gian -- Time Includes the current scene time and render frame at HH:MM:SS.FF . -- Render Time Includes the render time. Khung Hình -- Frame Includes the frame number. Frame Range Includes the start and end frame numbers. Bộ Nhớ -- Memory Includes the peak memory usage. Hostname Includes the rendering machine's hostname . Máy Quay Phim -- Camera Includes the name of the active camera. Lăng Kính -- Lens Includes the name of the active camera's lens value. Cảnh -- Scene Includes the name of the active scene. Dấu Mốc -- Marker Includes the name of the last marker. Tên Tập Tin -- Filename Includes the filename of the blend-file. Strip Name Includes the name of the foreground sequence strip. Bộ Phối Hình -- Sequencer Siêu Dữ Liệu của Dải -- Strip Metadata Use metadata from the strips in the Sequencer. Ghi Chú -- Note Includes a custom note. It can be useful to use the Note field if you are setting up a render farm. Since you can script any information you like into it, such as an identifier for the render node or the job number. For details on stamping arbitrary values, see: this page . Burn into Image Add metadata as text to the render. Kích Thước Phông -- Font Size Set the size of the text. Text Color Set the color and alpha of the stamp text. Background Set the color and alpha of the color behind the text. Include Labels Displays the labels before the metadata text. For example, "Camera" in front of the camera name, etc.

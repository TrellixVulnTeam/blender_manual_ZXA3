Introduction The Workbench Engine is a render engine optimized for fast rendering during modeling and animation preview. It is not intended to be a render engine that will render final images for a project. Its primary task is to display a scene in the 3D Viewport when it is being worked on. While its not intended to be used for final renders, the Workbench render engine can be selected as the Render Engine in the Render properties. By default the 3D Viewport uses Workbench to shade and light objects. Shading settings can be tweaked in the 3D Viewport's Shading popover . Workbench supports assigning random colors to objects to make each visually distinct. Other coloring mechanisms also exist, including; materials, vertex colors, and textures. Workbench also has an X-ray mode to see through objects, along with cavity and shadow shading to help display details in objects. Workbench supports several lighting mechanisms including studio lighting and MatCaps.

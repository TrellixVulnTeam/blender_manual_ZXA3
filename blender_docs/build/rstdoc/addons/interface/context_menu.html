Dynamic Context Menu
Introduction Dynamic Context Menu provides access to many of the menus found in the 3D View header. It also packs a few features of its own, such as the Cursor menu. Access Operator Search, Tools, Animation, Quick Favorites, Modifiers, Collections and more. It is context sensitive for the 3D View’s modes and object sensitive based on object selected in Edit Mode. This add-on pairs well with the Dynamic Brush Menus .
Activation Open Blender and go to Preferences then the Add-ons tab. Click Interface then Dynamic Context Menu to enable the script. If you have trouble accessing the add-on in the interface: You may need to set the Preferences ‣ Keymaps ‣ Spacebar Action to Tools and save your preferences. clear Reference Category Interface Description Object and mode context sensitive pop-up menu.

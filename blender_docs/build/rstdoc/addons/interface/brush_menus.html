Dynamic Brush Menus Reference Category Interface Description Fast access to brushes & tools in Sculpt and Paint Modes. Location kbd Spacebar in Sculpt/Paint Modes File space_view3d_brush_menus folder Author Ryan Inch (Imaginer)
Installation This add-on is bundled with Blender. Open Blender and go to Preferences then the Add-ons tab. Click Interface then Dynamic Brush Menus to enable the script.

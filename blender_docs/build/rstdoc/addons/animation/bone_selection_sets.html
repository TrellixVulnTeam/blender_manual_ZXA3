Bone Selection Sets Reference Category Animation Description List of Bone sets for easy selection while animating. Location Properties ‣ Object Data (Armature) ‣ Selection Sets File bone_selection_sets.py Author Ines Almeida, Sybren A. Stuvel, Antony Riakiotakis, Dan Eicher
Installation This add-on is bundled with Blender. Open Blender and go to Preferences then the Add-ons tab. Click Animation then Bone Selection Sets to enable the script.

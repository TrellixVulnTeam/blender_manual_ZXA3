Tri-Lighting
Introduction This add-on creates a simple three point studio style lighting set up.
Activation Open Blender and go to Preferences then the Add-ons tab. Click Lighting then Tri-Lighting to enable the script.
Interface Located in the 3D View ‣ Add ‣ Light menu .
Usage Select the object to point the lights at. 3D View ‣ Add ‣ Light menu ‣ 3 Point Lights Adjust settings in the Tri-Lighting Creator Adjust Last Operation panel. The created lights are pointed at and locked to the active object using a Track To constraint. In the Properties editor ‣ Light tab you can further edit the properties of your lights. Reference Category Lighting Description Add three point lighting to the selected or active object.

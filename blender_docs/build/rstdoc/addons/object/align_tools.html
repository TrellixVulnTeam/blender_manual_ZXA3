Align Tools
Introduction Use to align the location, rotation or scale of selected objects to the active object on the specified axes. Please see the old Wiki for the archived original docs.
Activation Open Blender and go to Preferences then the Add-ons tab. Click Interface then Align Tools to enable the script.

Is Key Free
Introduction Primarily used to check the availability of hotkeys and the editor types they are associated with.
Activation Open Blender and go to Preferences then the Add-ons tab. Click Development then Is key Free to enable the script.
Instructions Search All the registered keys associated with the pattern entered in the Key field and selected modifier shortcuts will be displayed. Click the Search icon to list the associated keys. Quick Type Alternatively, a list with free shortcuts is available for quick access of keys from the Quick Type selector. List All Shortcuts Print a list of all used shortcuts to the Text editor. Reference Category Development Description Find free shortcuts, inform about used and print a key list.

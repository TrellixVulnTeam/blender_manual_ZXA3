Relax Reference Category Mesh Description Relax the selected vertices while retaining the shape. Location 3D View ‣ Edit Mode Context Menu ‣ Relax File mesh_relax.py Author Fabian Fricke
Installation This add-on is bundled with Blender. Open Blender and go to Preferences then the Add-ons tab. Click Mesh then Relax to enable the script.

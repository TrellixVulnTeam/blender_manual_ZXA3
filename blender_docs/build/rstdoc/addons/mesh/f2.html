F2 Reference Category Mesh Description Extends the 'Make Edge/Face' functionality. Location Mesh Edit Mode :kbd: F File mesh_f2.py Author Bart Crouch, Alexander Nedovizin, Paul Kotelevets (concept design), Adrian Rutkowski
Installation This add-on is bundled with Blender. Open Blender and go to Preferences then the Add-ons tab. Click Mesh then F2 to enable the script.

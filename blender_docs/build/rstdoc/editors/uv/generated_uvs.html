UV Sinh Tạo -- Generated UVs
Tính Chất -- Properties Tham Chiếu -- Reference Chế Độ -- Mode Toàn Bộ các Chế Độ -- All Modes Bảng -- Panel Trình Biên Soạn Tính Chất (Properties editor) ‣ Dữ Liệu Vật Thể (Object Data) ‣ Không Gian Chất Liệu (Texture Space) These are settings of the texture space used by generated texture mapping. The visualization of the texture space can be activated in the Viewport Display . Không Gian Chất Liệu Tự Động -- Auto Texture Space Adjusts the active object's texture space automatically when transforming the object. Vị Trí, Kích Thước -- Location, Size If the texture space is not calculated automatically then you can define the location and size of the texture space relative to the base object. These can also be adjusted from the 3D View, see Editing for more information. Khung Lưới Chất Liệu -- Texture Mesh Use another mesh for texture indices, the vertex of the two objects must be perfectly aligned. otherwise the UV map will be distorted. Note that, this is only for mesh objects. Dùng UV để Ánh Xạ -- Use UV for Mapping Use UV values as generated texture coordinates. Note that, this is only for curve objects. Xứng Hợp với Không Gian Chất Liệu -- Match Texture Space Modifies the Location and Size to match the objects bounding box. This disables Auto Texture Space. Note that, this is only for curve objects. is Match Texture Space the same thing as Auto Texture Space?
Biên Soạn -- Editing Tham Chiếu -- Reference Chế Độ -- Mode Chế Độ Vật Thể và Chế Độ Biên Soạn -- Object Mode and Edit Mode Trình Đơn -- Menu Object ‣ Transform ‣ Scale/Move Texture Space To modify the texture space from the 3D View, enable Edit Texture Space while transforming an object.
Truy Cập -- Accessing The automatically calculated UV map can be accessed by an object's material through the Generated output of the Nút Tọa Độ Chất Liệu -- Texture Coordinate Node . This output can then be used to map any texture onto an object. Generated texture spaces do not have rotation support, to overcome this, a Nút Ánh Xạ -- Mapping Node can be used to rotate the UV map.

Lựa Chọn -- Selecting Selection tools are available in the Select Menu in the header, and the shortcuts listed below:
Trình Đơn -- Menu Box Select Click and drag to box select UV coordinates. Alternatively, use B to start box selection. See Select Box . Box Select Pinned Use the box lasso to select only pinned UV coordinates Ctrl-B . Chọn bằng Vòng Tròn -- Circle Select See Select Circle . Chọn/Bỏ Chọn Toàn Bộ -- Select/Deselect All Selects or deselects all UV coordinates A . Đảo Nghịch -- Inverse Inverts the current selection Ctrl-I . Chọn cái Đã Ghim Lại -- Select Pinned Selects all pinned UVs Shift-P . See Pinning. Chọn Cái Đã Kết Nối -- Select Linked This operator selects all UVs that are connected to currently selected UVs Ctrl-L . This works similarly to the tools in 3D View. Thêm/Bớt -- More/Less Ctrl-NumpadPlus , Ctrl-NumpadMinus Expands/Contracts the selection to/from the adjacent elements of the selection type. Phân Tách Lựa Chọn Y -- Select Split Y Cuts apart the selected UVs from the map. Only those UVs which belong to fully selected faces remain selected. As the name implies, this is particularly useful to unlink faces and move them elsewhere. The hotkey is analogous to the mesh Split tool. Select Overlap Selects any UVs that are extended over other UVs while also selecting any underlying UVs.
Tiêu Đề -- Header Đồng Bộ Hóa Lựa Chọn -- Sync Selection Turning on the Sync Selection button causes selection of components in the 3D View to sync with their corresponding elements in the UV editor. If off only the selected faces are displayed in the UV editor. These two modes have very different results when transforming components in the UV editor.
Chế Độ Lựa Chọn -- Selection Modes Select Modes dependent on the Sync Selection.
Tắt Đồng Bộ Hóa Chọn Lựa -- Sync Selection Off Điểm Đỉnh -- Vertex Select individual vertices. Cạnh -- Edge Select edges. Bề Mặt -- Face Select faces. Hải Đảo -- Island Select contiguous groups of faces. Chế Độ Lựa Chọn Dính Kết -- Sticky Selection Mode This selector lets you enable automatic additional selection. Cùng Điểm Đỉnh -- Shared Vertex Selects UVs that share a mesh vertex, even if they are in different UV locations. Cùng Vị Trí -- Shared Location Selects UVs that are in the same UV location and share a mesh vertex. Tắt -- Disabled Disables Sticky Selection. When you move a UV in this mode, each face owns its own UVs, allowing them to be separated.
Bật Đồng Bộ Hóa Chọn Lựa -- Sync Selection On When selecting UVs or Edges, it behave like Shared Vertex mode above. When selecting Faces, it behaves as in Disabled Stick Selection above. Điểm Đỉnh -- Vertex Cạnh -- Edge Bề Mặt -- Face

Quản Lý Ánh Xạ UV -- Managing UV Maps After you finish editing a UV map, you may need to create additional maps on the same object, or transfer a UV map to another mesh.
Thuyên Chuyển Ánh Xạ UV -- Transferring UV Maps You can copy a UV map from one mesh to another Mesh provided both meshes have the same geometry/vertex order. This is useful for example when you want to recreate a UV map from an earlier version of your model with intact UVs.
Quy Trình Làm Việc -- Workflow Select the target mesh (to which you want to copy the UV map). Shift select the source mesh (that contains the intact UV map). Trình đơn Vật Thể (Object menu) ‣ Tạo Kết Nối... (Make Links...) ‣ Thuyên Chuyển Bố Trí UV (Transfer UV Layouts) (Shortcut: Ctrl-L ...). The target Mesh will now have a UV map that matches the original mesh.
Nhiều Ánh Xạ UV -- Multiple UV Maps You are not limited to one UV map per mesh. You can have multiple UV maps for parts of the mesh by creating new UV maps. This can be done by clicking the Add button next to UV maps list (in Object Data tab in the Properties Editor) and unwrapping a different part of the mesh. UV maps always include the whole mesh. Bảng Ánh Xạ UV -- UV Maps Panel In the Mesh tab the UV maps panel contains a List view that lists the UV maps created for this mesh. The selected map is displayed in the UV Editor. Cho Kết Xuất -- Active Render Click the camera icon to enable that UV texture for rendering. If no other map is explicitly specified.

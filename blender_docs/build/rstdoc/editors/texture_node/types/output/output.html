Nút Đầu Ra -- Output Node This node contains the result of the node texture. Multiple output nodes can exist in a node texture, however, only one of them is active. The active one is set in the Texture Panel in the Output selector.
Đầu Vào -- Inputs Màu Sắc -- Color The color data that the texture renders. Pháp Tuyến -- Normal The normal map that the texture will output.
Tính Chất -- Properties Đường dẫn tập tin -- File path Output ID.
Đầu Ra -- Outputs Nút này không có đầu ra.

Nút Chất Liệu -- Texture Node The texture node can be used to load another node-based or non-node-based texture.
Đầu Vào -- Inputs These two colors can be used to remap a grayscale texture. Màu 1 -- Color 1 White Level. Màu 2 -- Color 2 Black Level.
Tính Chất -- Properties Chất Liệu -- Texture The texture could be selected from a list of textures available in the current blend-file or link in textures. The textures themselves could not be edited in this node, but in the Texture panel.
Đầu Ra -- Outputs Màu Sắc -- Color Đầu ra màu sắc chuẩn.

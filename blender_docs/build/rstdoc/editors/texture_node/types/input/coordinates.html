-- Coordinates Node
Đầu Vào -- Inputs Nút này không có đầu vào.
Tính Chất -- Properties Nút này không có tính chất nào cả.
Đầu Ra -- Outputs Tọa Độ -- Coordinates The Coordinates node outputs the geometry local coordinates, relative to its bounding box as RGB colors: Red channel corresponds to X value. Green channel corresponds to Y value. Blue channel corresponds to Z value.

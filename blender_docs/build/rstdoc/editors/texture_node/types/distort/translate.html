Nút Dịch Chuyển -- Translate Node Translate the texture coordinates of an image or texture.
Đầu Vào -- Inputs Màu Sắc -- Color Đầu vào hình ảnh tiêu chuẩn. Dịch Chuyển -- Offset The amount to offset the coordinates in each of the three axes.
Tính Chất -- Properties Nút này không có tính chất nào cả.
Đầu Ra -- Outputs Màu Sắc -- Color Đầu ra tiêu chuẩn của hình ảnh.

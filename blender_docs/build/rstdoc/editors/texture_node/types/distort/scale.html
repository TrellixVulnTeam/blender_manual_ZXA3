Nút Tỷ Lệ -- Scale Node Scale the texture coordinates of an image or texture.
Đầu Vào -- Inputs Màu Sắc -- Color Đầu vào hình ảnh tiêu chuẩn. Tỷ Lệ -- Scale The amount to scale the coordinates in each of the three axes.
Tính Chất -- Properties Nút này không có tính chất nào cả.
Đầu Ra -- Outputs Màu Sắc -- Color Đầu ra tiêu chuẩn của hình ảnh.

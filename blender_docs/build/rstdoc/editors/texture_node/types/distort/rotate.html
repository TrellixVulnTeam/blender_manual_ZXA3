Nút Xoay Chiều -- Rotate Node Rotate the texture coordinates of an image or texture.
Đầu Vào -- Inputs Màu Sắc -- Color Đầu vào hình ảnh tiêu chuẩn. Số Lần Xoay -- Turns The number of times to rotate the coordinates 360 degrees about the specified axis. Trục -- Axis The axis to rotate the mapping about.
Tính Chất -- Properties Nút này không có tính chất nào cả.
Đầu Ra -- Outputs Màu Sắc -- Color Đầu ra tiêu chuẩn của hình ảnh.

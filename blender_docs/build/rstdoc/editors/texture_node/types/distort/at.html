-- At Node Returns the color of a texture at the specified coordinates.
Đầu Vào -- Inputs Chất Liệu -- Texture Đầu vào hình ảnh tiêu chuẩn. Tọa Độ -- Coordinates The point at which to sample the color. For images, the space is between -1 and 1 for X and Y. If the coordinates are not spatially varying, the node will return a single color.
Tính Chất -- Properties Nút này không có tính chất nào cả.
Đầu Ra -- Outputs Chất Liệu -- Texture Đầu ra tiêu chuẩn của hình ảnh.

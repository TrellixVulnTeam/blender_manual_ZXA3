-- Checker Node The checker node creates a checkerboard pattern.
Đầu Vào -- Inputs -- Color 1, Color 2 Image inputs setting the color of the squares. Kích Thước -- Size The scale of the checker pattern.
Tính Chất -- Properties Nút này không có tính chất nào cả.
Đầu Ra -- Outputs Màu Sắc -- Color Đầu ra tiêu chuẩn của hình ảnh.

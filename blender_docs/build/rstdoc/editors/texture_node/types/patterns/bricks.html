-- Bricks Node The Bricks node creates a brick like pattern.
Đầu Vào -- Inputs -- Bricks 1, Bricks 2 Sets the color range of the bricks. Brick colors are chosen randomly between these two colors. Vữa -- Mortar Sets the mortar color, in between the bricks. Độ Dày -- Thickness Sets the thickness of the mortar. Thiên Lệch -- Bias The bias of randomly chosen colors, between (-1 to 1). -1 Makes all bricks Color 1, and a value of 1 makes them all Color 2. Chiều Rộng của Gạch -- Brick Width Sets the horizontal size of all the bricks. Chiều Cao của Hàng -- Row Height Sets the vertical size of all the bricks.
Tính Chất -- Properties Dịch Chuyển -- Offset The relative offset of the next row of bricks. Tần Số -- Frequency Offset every N rows. The brick pattern offset repeats every N rows. Nén ép -- Squash Scales the bricks in every N rows by this amount. Tần Số -- Frequency Squash every N rows.
Đầu Ra -- Outputs Màu Sắc -- Color Đầu ra màu sắc chuẩn.

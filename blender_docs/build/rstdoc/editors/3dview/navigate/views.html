Góc Nhìn theo Ngữ Cảnh -- Contextual Views The 3D View has several "contextual view" modes that can be set for a particular 3D View. These views can change how the overall 3D View looks or how you interact with objects.
Góc Nhìn Toàn Cầu/Địa Phương -- View Global/Local Tham Chiếu -- Reference Chế Độ -- Mode Tổng Thể -- All modes Trình Đơn -- Menu View ‣ Local View ‣ Toggle Local View Phím Tắt -- Hotkey NumpadSlash , Slash Global view shows all of the 3D objects in the scene. Local view isolates the selected object or objects, so that they are the only ones visible in the viewport. This is useful for working on objects that are obscured by other ones, or to speed up the viewport performance in heavy scenes. You can toggle between Global and Local View by selecting the option from the View Menu or using the shortcut NumpadSlash .

These notes cover changes in local view which are not immediately obvious. Con Trỏ 3D -- 3D Cursor In local view the 3D cursor is not locked to the scene. Instead, each view has an independent cursor location. Tầng Lớp -- Layers Local view bypasses layers, using only the selected objects when entering local view. Although new objects may be added while in local view. It's also possible to send objects out of local view, using Vật Thể (Object) ‣ Di Chuyển Vật Thể ra khỏi Góc Nhìn Địa Phương (Move Objects out of Local View) , which can be useful to further isolate a selection. Accidentally pressing NumpadSlash can happen rather often if you are new to Blender, so if a bunch of the objects in your scene seem to have mysteriously vanished, try turning off local view.
Remove from Local View Tham Chiếu -- Reference Chế Độ -- Mode Tổng Thể -- All modes Trình Đơn -- Menu View ‣ Local View ‣ Remove from Local View Phím Tắt -- Hotkey M Objects can be removed from Local View by selecting them and using the Remove from Local View operator. This will move the selected object back to global view and all other objects will remain in local view. If the last remaining object is removed, the local view will be left empty and you will have to exit local view to see any objects.
Bốn Góc Nhìn -- Quad View Tham Chiếu -- Reference Chế Độ -- Mode Tổng Thể -- All modes Trình Đơn -- Menu Góc Nhìn (View) ‣ Bật/Tắt 4 Góc Nhìn (Toggle Quad View) Bảng -- Panel Sidebar region ‣ Display ‣ Toggle Quad View Phím Tắt -- Hotkey Ctrl-Alt-Q Toggling Quad View will split the 3D View into four views: Three Orthographic "side views" and one Camera / User View . This view will allow you to instantly see your model from a number of view points. In this arrangement, you can zoom and pan each view independently but you cannot rotate the view. Shortcuts for all views at once: -- Shortcuts for all views at once: View All Ctrl-Home View Selected Ctrl-NumpadPeriod Quad View is different from splitting the area and aligning the view manually. In Quad View, the four views are still part of a single 3D View. So they share the same display options and layers.
Tùy Chọn -- Options These options can be found in Sidebar region ‣ Display . Khóa -- Lock If you want to be able to rotate each view, you can uncheck the Locked option. Hình Hộp -- Box Syncs the view position between side views. Đoạn Phim -- Clip Clip objects based on what is visible in other side views.

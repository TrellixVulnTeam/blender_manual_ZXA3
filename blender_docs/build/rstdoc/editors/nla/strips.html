Dải Phim -- Strips
Thể Loại -- Types There are four kinds of strips: Action, Transition, Sound clip and Meta.
Dải Hành Động -- Action Strips An Action Strip is a container of keyframe data of an action. Any action used by the NLA first must be turned into an Action strip. This is done so by clicking the Push Down action button see above. Alternatively, you can go to Cộng Thêm (Add) ‣ Dải Hành Động (Action Strip) .
Dải Chuyển Cảnh -- Transition Strips Transitions interpolate between Actions. They must be placed in between other strips. Select two or more strips on the same track, and go to Cộng Thêm (Add) ‣ Chuyển Cảnh/Tiếp (Transition) .
Dải Âm Thanh -- Sound Clip Strips Controls when a speaker plays a sound clip. Cộng Thêm (Add) ‣ Đoạn Âm Thanh (Sound Clip) .
Dải Siêu -- Meta Strips Meta strips group strips together as a whole, so you can move them as one. If you find yourself moving a lot of strips together, you can group them into a Meta strip. A Meta strip can be moved and duplicated like a normal strip. Tham Chiếu -- Reference Trình Đơn -- Menu Cộng Thêm (Add) ‣ Cộng Thêm Dải Siêu (Add Meta-Strips) Phím Tắt -- Hotkey Shift-G
Select two or more strips. Combine them into a meta strip.
A Meta strip still contains the underlying strips. You can ungroup a Meta strip. Tham Chiếu -- Reference Trình Đơn -- Menu Cộng Thêm (Add) ‣ Xóa các Dải Siêu (Remove Meta-Strips) Phím Tắt -- Hotkey Alt-G
Biên Soạn -- Editing
Bắt Đầu Hiệu Chỉnh Hành Động trong Dải Phim -- Start Tweaking Strips Action Tham Chiếu -- Reference Trình Đơn -- Menu Biên Soạn (Edit) ‣ Bắt Đầu Tinh Chỉnh Hành Động trong Dải Phim (Start Tweaking Strips Action) Phím Tắt -- Hotkey Tab The contents of Action strips can be edited, but you must be in Tweak Mode to do so. The keyframes of the action can then be edited in the Dope Sheet.
Strip in NLA mode. Strip in Tweak mode.
When you finished editing the strip, simply go to Biên Soạn (Edit) ‣ Tinh Chỉnh Dải Hành Động (Tweaking Strips Action) or press Tab .
Bắt Đầu Biên Soạn Hành Động đã Cất Giấu -- Start Editing Stashed Action Tham Chiếu -- Reference Trình Đơn -- Menu Biên Soạn (Edit) ‣ Bắt Đầu Biên Soạn Hành Động đã Cất Giấu (Start Editing Stashed Action) Phím Tắt -- Hotkey Shift-Tab It will enter and exit Tweak Mode as usual, but will also make sure that the action can be edited in isolation (by flagging the NLA track that the action strip comes from as being "solo"). This is useful for editing stashed actions, without the rest of the NLA Stack interfering.
Sao Chép -- Duplicate Tham Chiếu -- Reference Trình Đơn -- Menu Biên Soạn (Edit) ‣ Sao Chép (Duplicate) Phím Tắt -- Hotkey Shift-D Creates a new instance of the selected strips with a copy of the action.
Bản Sao Chép Kết Nối -- Linked Duplicate Tham Chiếu -- Reference Trình Đơn -- Menu Biên Soạn (Edit) ‣ Bản Sao Chép Kết Nối (Linked Duplicate) Phím Tắt -- Hotkey Alt-D The contents of one Action strip can be instanced multiple times. To instance another strip, select a strip, go to Biên Soạn (Edit) ‣ Bản Sao Chép Kết Nối (Linked Duplicate) . It will uses the same action as the selected strips. Now, when any strip is tweaked, the others will change too. If a strip other than the original is tweaked, the original will turn to red.
Biến Thành Đơn Người Dùng -- Make Single User Tham Chiếu -- Reference Trình Đơn -- Menu Biên Soạn (Edit) ‣ Biến Thành Đơn Người Dùng (Make Single User) Phím Tắt -- Hotkey U This tool ensures that none of the selected strips use an action which is also used by any other strips. (dev) NOTE: This does not recursively go inside meta's, so care is still advised in that case.

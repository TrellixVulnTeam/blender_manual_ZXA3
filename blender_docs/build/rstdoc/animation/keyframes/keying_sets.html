Bộ Khóa -- Keying Sets (alt) Keying Sets are a set of keyframe channels. Keying Sets are a collection of properties. They are used to record multiple properties at the same time. Now when you press I in the 3D View, Blender will add keyframes for all the properties in the active keying set. There are some built-in Keying Sets and, also custom Keying Sets called Absolute Keying Sets . To select and use a Keying Set, set the Active Keying Set in the Keying popover in the Timeline Header, or the Keying Set panel, or press Shift-Ctrl-Alt-I in the 3D View.

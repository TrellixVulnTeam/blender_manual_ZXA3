Bộ Màu Sắc -- Color Set TODO2.8 .. figure:: /images/animation_armatures_properties_bone-groups_color-list.png The Bone Color Set selector and the color fields. You can assign a "color theme" to a group (each bone will have these colors). Remember you have to enable the Colors checkbox ( Display panel) to see these colors. Bộ Màu Sắc của Xương -- Bone Color Set A select menu. Default Colors : The default (gray) colors. nn - Theme Color Set : One of the twenty Blender presets by the theme. Custom Set : A custom set of colors, which is specific to each group. Pháp Tuyến -- Normal The first color field is the color of unselected bones. Cái Được Chọn -- Selected The second color field is the outline color of selected bones. Đang/Hoạt Động -- Active The third color field is the outline color of the active bone. As soon as you alter one of the colors, it is switched to the Custom Set option.
-- Assign and Select In the 3D Views, using the Tư Thế (Pose) ‣ Nhóm Xương (Bone Groups) menu entries, and/or the Bone Groups pop-up menu Ctrl-G , you can: Chỉ Định -- Assign Assigns the selected bones to the active bone group. It is important to note that a bone can only belong to one group. Xóa -- Remove Removes the selected bones from the active bone group. Chọn -- Select Selects the bones in the active bone group. Hủy Chọn -- Deselect Deselects the bones in the active bone group. A single bone can be assigned to a group in the Relations panel . Bones belonging to multiple groups is possible with the Selection Sets add-on.

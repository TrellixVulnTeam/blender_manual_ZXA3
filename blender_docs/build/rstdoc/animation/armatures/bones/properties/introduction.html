Biến Hóa -- Transform Tham Chiếu -- Reference Chế Độ -- Mode Edit Mode and Pose Mode Bảng -- Panel Xương (Bone) ‣ Biến Hóa (Transform) TODO2.8 .. list-table:: * - .. figure:: /images/animation_armatures_bones_properties_introduction_transform-panel-edit.png The Transform panel (Edit Mode). - .. figure:: /images/animation_armatures_bones_properties_introduction_transform-panel-pose.png The Transform panel (Pose Mode). When in Edit Mode you can use this panel to control position and roll of individual bones. Whereas in Pose Mode you can only set location for the main bone, and you can now set rotation and scale. In addition, in Pose Mode it is possible to restrict changes in position, rotation and scale by axis on each bone in the armature.

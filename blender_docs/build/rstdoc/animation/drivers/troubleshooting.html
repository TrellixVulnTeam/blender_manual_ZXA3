Biểu Thức Tập Lệnh -- Scripted Expression By default Blender will restrict execution of Python scripts. If using a Scripted Expression Driver Type that doesn't follow the Simple Expressions subset, you will have to open the file as Trusted Source , or set Auto Run Python Scripts in Preferences ‣ Save & Load ‣ Blender Files .
The Trusted Source checkbox in the File Browser. The Auto Run Python Scripts checkbox in the Preferences.

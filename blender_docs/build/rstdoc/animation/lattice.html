Giàn Lưới -- Lattice Lattice -- or commonly called deformation cage outside of Blender. A lattice consists of a three-dimensional non-renderable grid of vertices. Its main use is to apply a deformation to the object it controls with a Bộ Điều Chỉnh Giàn Lưới -- Lattice Modifier . If the object is parented with Lattice Deform a Lattice Modifier is automatically applied.
Biên Soạn -- Editing Đảo Chiều (Không Biến Dạng) -- Flip (Distortion Free) Mirrors the vertices displacement from their base position. U, V, W Sắp Đều -- Make Regular Resets the whole lattice to a regular grid, where the cells are scaled to one cubic unit.

Sử Dụng -- Usage TODO2.8 Maybe update the images (color & style) The lattice should be scaled and moved to fit around your object in Object Mode. Any scaling applied to the object in Edit Mode will result in the object deforming. This includes applying its scale with Ctrl-A as this will achieve the same result as scaling the lattice in Edit Mode, and therefore the object.

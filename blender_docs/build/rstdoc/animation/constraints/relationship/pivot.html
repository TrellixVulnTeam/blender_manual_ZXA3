Ràng Buộc Điểm Tựa -- Pivot Constraint The Pivot constraint allows the owner to rotate around a target object. It was originally intended for pivot joints found in humans e.g. fingers, feet, elbows, etc.

Dùng Điểm Tựa Khi -- Pivot When Always, Z Rot, Y Rot... Nội dung cần viết thêm.

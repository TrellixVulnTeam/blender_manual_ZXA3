Blender's Directory Layout This page documents the different directories used by Blender (which can be helpful for troubleshooting) . There are three different directories Blender may use, their exact locations are platform dependent. ĐỊA PHƯƠNG -- LOCAL Vị trí của cấu hình và dữ liệu sử dụng trong khi phần mềm đang hoạt động (dành cho gói độc lập). NGƯỜI DÙNG -- USER Location of configuration files (typically in the user's home directory). HỆ THỐNG -- SYSTEM Vị trí của dữ liệu sử dụng thời gian chạy (lúc đang hoạt động) để cài đặt trên toàn hệ thống (có thể chỉ đọc mà thôi). Để cài đặt hệ thống, cả hai thư mục SYSTEM và USER đều cần thiết. For locally extracted Blender distributions, the user configuration and run-time data are kept in the same subdirectory, allowing multiple Blender versions to run without conflict, ignoring the USER and SYSTEM files. This requires you to create a folder named config in the LOCAL directory.
Những Đường Dẫn Phụ Thuộc Nền Tảng (Hệ Điều Hành) -- Platform Dependent Paths Dưới đây là các vị trí mặc định cho mỗi hệ thống:
Linux ĐỊA PHƯƠNG -- LOCAL ./2.81/ NGƯỜI DÙNG -- USER $HOME/.config/blender/2.81/ HỆ THỐNG -- SYSTEM /usr/share/blender/2.81/ The path ./2.81/ is relative to the Blender executable and is used for self-contained bundles distributed by official blender.org builds. Đường dẫn USER ( NGƯỜI DÙNG ) sẽ sử dụng biến môi trường $XDG_CONFIG_HOME nếu nó được thiết lập: $XDG_CONFIG_HOME/blender/2.81/
Hệ Điều Hành Macintosh của Apple -- macOS ĐỊA PHƯƠNG -- LOCAL ./2.81/ NGƯỜI DÙNG -- USER /Users/$USER/Library/Application Support/Blender/2.81/ HỆ THỐNG -- SYSTEM /Library/Application Support/Blender/2.81/ macOS stores the Blender binary in ./Blender.app/Contents/MacOS/Blender . The local path to data and config is: ./Blender.app/Contents/Resources/2.81/
Windows ĐỊA PHƯƠNG -- LOCAL .\2.81\ NGƯỜI DÙNG -- USER %USERPROFILE%\AppData\Roaming\Blender Foundation\Blender\2.81\ HỆ THỐNG -- SYSTEM %USERPROFILE%\AppData\Roaming\Blender Foundation\Blender\2.81\
Bố Trí Đường Dẫn -- Path Layout Đây là cách bố trí đường dẫn được sử dụng trong các thư mục được mô tả ở trên. Trong đó ./config/startup.blend có thể nằm tại ~/.blender/|BLENDER_VERSION|/config/startup.blend chẳng hạn. ./autosave/ ... Autosave blend-file location. (Windows only, temp directory used for other systems.) Thứ tự lùng tìm: LOCAL, USER . ./config/ ... Mặc định & thông tin về phiên sử dụng. Thứ tự lùng tìm: LOCAL, USER . ./config/startup.blend Tập tin mặc định để nạp nạp vào bộ nhớ khi khởi động. ./config/userpref.blend Các tùy chọn mặc định để nạp vào bộ nhớ khi khởi động. ./config/bookmarks.txt Các dấu trang cho Trình Duyệt Tập Tin (Những thư mục được đánh dấu, ghi lại). ./config/recent-files.txt Trình đơn liệt kê tập tin sử dụng gần đây nhất ./datafiles/ ... Những tập tin sử dụng trong khi phần mềm đang chạy Thứ tự lùng tìm: LOCAL, USER, SYSTEM . ./datafiles/locale/{language}/ Các tập tin ngôn ngữ tĩnh biên dịch trước dành cho sự phiên dịch Giao Diện Người Dùng ./scripts/ ... Các tập lệnh Python cho giao diện người dùng và các công cụ. Thứ tự lùng tìm: LOCAL, USER, SYSTEM . ./scripts/addons/*.py Python add-ons which may be enabled in the Preferences include import/export format support, render engine integration and many handy utilities. ./scripts/addons/modules/*.py Modules for add-ons to use (added to Python's sys.path ). ./scripts/addons_contrib/*.py Một thư mục cho các trình bổ sung được sử dụng cho các tiện ích bổ sung do cộng đồng quản lý (phải được tạo một cách thủ công). ./scripts/addons_contrib/modules/*.py Modules for addons_contrib to use (added to Python's sys.path ). ./scripts/modules/*.py Các mô đun Python chứa các API cốt lõi của chúng ta và các hàm tiện ích cho các tập lệnh khác để nhập khẩu (đã được thêm vào `` sys.path`` của Python). ./scripts/startup/*.py Các tập lệnh được nhập tự động khi khởi động. ./scripts/presets/{preset}/*.py Những sắp đặt sẵn được sử dụng để lưu trữ các cài đặt do người dùng xác định cho vải, các định dạng kết xuất, v.v. ./scripts/templates_py/*.py Các tập lệnh mẫu có thể được truy cập từ Trình Biên Soạn Văn Bản (Text Editor) ‣ Khuôn Mẫu (Templates) ‣ Python . ./scripts/templates_osl/*.osl Ví dụ về các bộ tô bóng OSL, có thể được truy cập từ Trình Biên Soạn Văn Bản (Text Editor) ‣ Khuôn Mẫu (Templates) ‣ Ngôn Ngữ Tô Bóng Mở (Open Shading Language) . ./python/ ... Bản phân phối có kèm Python. Thứ tự lùng tìm: LOCAL, SYSTEM .
Thư Mục Tạm Thời -- Temporary Directory Thư mục tạm thời được sử dụng để lưu trữ các tập tin khác nhau trong thời gian phần mềm đang hoạt động (bao gồm các lớp kết xuất, bộ nhớ đệm cache vật lý, bộ nhớ sao chép và bản ghi sự cố). Thư mục tạm thời được chọn dựa trên mức độ ưu tiên sau: Tùy Chọn Người Dùng (xin xem mục File Paths ). Environment variables ( TEMP on Windows, TMP & TMP_DIR on other platforms). Thư mục /tmp/ .

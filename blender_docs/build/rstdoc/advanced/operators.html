Thao Tác/Toán Tử -- Operators
Trình Đơn Hiệu Chỉnh Lỗi -- Debug Menu Tham Chiếu -- Reference Chế Độ -- Mode Toàn Bộ các Chế Độ -- All Modes Phím Tắt -- Hotkey Ctrl-Alt-D This operator brings up a menu to set Blender into a certain debug mode. See the source code for a description of what each value does. Developers can search the code for G.debug_value to find other possible uses for this operator. Additional debug options are available by launching Blender in debug mode or setting bpy.app.debug = True .
Vẽ Lại Đồng Hồ -- Redraw Timer Tham Chiếu -- Reference Chế Độ -- Mode Toàn Bộ các Chế Độ -- All Modes Phím Tắt -- Hotkey Ctrl-Alt-T This operator brings up a menu with a list of tests to benchmark UI render times along with undo/redo functions.
Thống Kê về Bộ Nhớ -- Memory Statistics Tham Chiếu -- Reference Chế Độ -- Mode --debug-memory This operator which can be found by searching "Memory Statistics" from the Operator Search will print useful information about memory objects, their size and user count. To fully use this operator run Blender from the console with --debug-memory .
Quan Hệ Phụ Thuộc -- Dependency Relations Tham Chiếu -- Reference Chế Độ -- Mode --debug-depsgraph This operator which can be found by searching "Dependency Relations" from the Operator Search will print the relations for every scene and object in the blend-file. This operator will only work if Blender is started from the console with --debug-depsgraph .

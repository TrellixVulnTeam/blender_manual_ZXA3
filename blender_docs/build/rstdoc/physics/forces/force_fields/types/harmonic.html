Điều Hòa -- Harmonic Tham Chiếu -- Reference Bảng -- Panel Vật Lý (Physics) ‣ Trường Lực (Force Fields) Thể Loại -- Type Điều Hòa -- Harmonic In a Harmonic force field, the source of the force field is the zero point of a harmonic oscillator (spring, pendulum). If you set the Damping parameter to 1, the movement is stopped in the moment the object is reached. This force field is really special if you assign it to particles.
Tùy Chọn -- Options TODO2.8: .. figure:: /images/physics_forces_force-fields_types_harmonic_panel.png UI for a Harmonic force field. Chiều Dài Nghỉ -- Rest Length Controls the rest length of the harmonic force. Bội Số Lò Xo -- Multiple Springs Causes every point to be affected by multiple springs. Normally every particle of the field system influences every particle of the target system. Not with Harmonic ! Here every target particle is assigned to a field particle. So particles will move to the place of other particles, thus forming shapes.

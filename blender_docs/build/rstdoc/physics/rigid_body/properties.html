-- Rigid Body Properties
Thân Cứng -- Rigid Body Tham Chiếu -- Reference Bảng -- Panel Vật Lý (Physics) ‣ Thân Cứng (Rigid Body) TODO2.8: .. figure:: /images/physics_rigid-body_properties_panel.png Default rigid body panel. Thể Loại -- Type Role of the rigid body in the simulation. Active objects can be simulated dynamically, passive object remain static. Đang/Hoạt Động -- Active Object is directly controlled by simulation results. The possibility to select this type also available with Add Active button in the Physics tab of the Toolbar. Thụ Động -- Passive Object is directly controlled by animation system. Thus, this type is not available for Dynamics . The possibility to select this type also available with Add Passive button in the Physics tab of the Toolbar. Khối Lượng -- Mass Specifies how heavy the object is and "weights" irrespective of gravity. There are predefined mass preset available with the Calculate Mass button in the Physics tab of the Toolbar. Tính Toán Khối Lượng -- Calculate Mass Automatically calculate mass values for rigid body objects based on its volume. There are many useful presets available from the menu, listing real-world objects. Also you can have Custom mass material type, which is achieved by setting a custom density value (kg/m 3 ). Năng Động -- Dynamic Enables/disables rigid body simulation for object. Hoạt Họa -- Animated Allows the rigid body additionally to be controlled by the animation system.
Va Chạm Thân Cứng -- Rigid Body Collisions Tham Chiếu -- Reference Bảng -- Panel Physics ‣ Rigid Body ‣ Collisions TODO2.8: .. figure:: /images/physics_rigid-body_properties_collisions.png Rigid Body Collisions panel.
Hình Dạng Khối Va Đập -- Collision Shapes The Shape option determines the collision shape of the object. -- Primitive Shapes These are best in terms of memory/performance but do not necessarily reflect the actual shape of the object. They are calculated based on the object's bounding box. The center of gravity is always in the middle for now. Primitive shapes can be shown in the viewport by enabling Bounds in the Vật Thể (Object) ‣ Hiển Thị (Display) panel. Hình Hộp -- Box Box-like shapes (e.g. cubes), including planes (e.g. ground planes). The size per axis is calculated from the bounding box. Hình Cầu -- Sphere Sphere-like shapes. The radius is the largest axis of the bounding box. Vỏ Nhộng -- Capsule This points up the Z axis. Hình Trụ -- Cylinder This points up the Z axis. The height is taken from the Z axis, while the radius is the larger of the X or Y axes. Hình Nón -- Cone This points up the Z axis. The height is taken from the Z axis, while the radius is the larger of the X or Y axes. -- Mesh-Based Shapes These are calculated based on the geometry of the object so they are a better representation of the object. The center of gravity for these shapes is the object origin. Bao Lồi -- Convex Hull A mesh-like surface encompassing (e.g. shrink-wrap over) all vertices (best results with fewer vertices). A convex approximation of the object, has a good performance and stability. Khung Lưới -- Mesh Mesh consisting of triangles only, allowing for more detailed interactions than convex hulls. Allows to simulate concave objects, but is rather slow and unstable.
Khung Lưới Nguồn -- Mesh Source Users can now specify the mesh Source for Mesh bases collision shapes: Cơ Sở -- Base The base mesh of the object. Biến Dạng -- Deform Includes any deformations added to the mesh (shape keys, deform modifiers). Biến Dạng -- Deforming Mesh shapes can deform during simulation. Kết Thúc -- Final Includes all deformations and modifiers.
-- Surface Response Tham Chiếu -- Reference Bảng -- Panel Physics ‣ Rigid Body ‣ Surface Response Ma Sát -- Friction Resistance of object to movement. Specifies how much velocity is lost when objects collide with each other. Độ Nẩy Bật -- Bounciness Tendency of object to bounce after colliding with another (0 to 1) (rigid to perfectly elastic). Specifies how much objects can bounce after collisions. Nhóm Va Đập -- Collision Groups Allows rigid body collisions allocate on different groups (maximum 20).
Sensitivity Tham Chiếu -- Reference Bảng -- Panel Physics ‣ Rigid Body ‣ Sensitivity Lề -- Margin Threshold of distance near surface where collisions are still considered (best results when non-zero). The collision margin is used to improve the performance and stability of rigid bodies. Depending on the shape, it behaves differently: some shapes embed it, while others have a visible gap around them. The margin is embedded for these shapes: Hình Cầu -- Sphere Hình Hộp -- Box Vỏ Nhộng -- Capsule Hình Trụ -- Cylinder Convex Hull: Only allows for uniform scale when embedded. The margin is not embedded for these shapes: Hình Nón -- Cone Active Triangle Mesh Passive Triangle Mesh: Can be set to 0 most of the time.
Dynamics Tham Chiếu -- Reference Bảng -- Panel Physics ‣ Rigid Body ‣ Dynamics TODO2.8: .. figure:: /images/physics_rigid-body_properties_dynamics.png Rigid Body Dynamics panel. Used to control the physics of the rigid body simulation. This panel is available only for Active type of rigid bodies. Giảm Chấn -- Damping Dịch Chuyển -- Translation Amount of linear velocity that is lost over time. Xoay Chiều -- Rotation Amount of angular velocity that is lost over time. -- Deactivation Cho Phép Giải Hoạt -- Enable Deactivation Enable deactivation of resting rigid bodies. Allows object to be deactivated during the simulation (improves the performance and stability, but can cause glitches). Giải Hoạt Khi Bắt Đầu -- Start Deactivated Starts objects deactivated. They are activated on collision with other objects. Vận Tốc Tuyến Tính -- Linear Velocity Specifies the linear deactivation velocity below which the rigid body is deactivated and simulation stops simulating object. Vận Tốc Góc -- Angular Velocity Specifies the angular deactivation velocity below which the rigid body is deactivated and simulation stops simulating object.

Nút Ánh Xạ Giá Trị -- Map Value Node Nút Ánh Xạ Giá Trị được sử dụng để tỷ lệ hóa (scale), dịch chuyển (offset), và hạn định (clamp) các giá trị.
Đầu Vào -- Inputs Giá Trị -- Value Standard Value input. (Value refers to each vector in the set.)
Tính Chất -- Properties Dịch Chuyển -- Offset Hệ số được thêm vào giá trị đầu vào. Kích Thước -- Size Tỷ lệ hóa (nhân) giá trị đầu vào. Sử dụng Tối Thiểu, Tối Đa -- Use Minimum, Maximum Bật tính năng này lên để kích hoạt phép toán liên quan của chúng. Tối Thiểu, Tối Đa -- Min, Max Xác định một phạm vi giữa tối thiểu và tối đa để hạn định ( clamp ) giá trị đầu vào.
Đầu Ra -- Outputs Giá Trị -- Value Đầu ra tiêu chuẩn của giá trị.
Ví Dụ -- Example
Ánh Xạ Độ Sâu Z -- Z-Depth Map Điều này đặc biệt hữu ích khi mình muốn tạo hiệu ứng độ sâu trường ảnh, trong đó nút Ánh Xạ Giá Trị ( Map Value ) được sử dụng để ánh xạ giá trị Z (có thể là 20, 30 hoặc thậm chí là 500 tùy theo cảnh) sang phạm vi giữa (0 đến 1), thích hợp để kết nối với nút Làm Nhòe ( Blur ).
Nhân các Giá Trị -- Multiplying Values Nút Ánh Xạ Giá Trị ( Map Value ) cũng có thể được sử dụng để nhân giá trị lên thành một giá trị đầu ra mong muốn. Trong sơ đồ nhỏ ở bên phải, nút Thời Gian ( Time ) cho ra một giá trị trong phạm vi từ 0,0 đến 1,0, được phân tỷ lệ đồng đều trên 30 khung hình. Nút Ánh Xạ Giá Trị ( Map Value ) đầu tiên nhân giá trị đầu vào lên 2, dẫn đến giá trị của đầu ra được phân tỷ lệ từ 0,0 đến 2,0 trên 30 khung hình. Nút Ánh Xạ Giá Trị ( Map Value ) thứ hai khấu trừ 1 khỏi giá trị đầu vào, tạo ra các giá trị hoạt động trong khoảng từ (-1,00 đến 1) và nhân nó với 150 để tạo thành giá trị đầu ra trong khoảng từ (-150 đến 150) trong trình tự 30 khung hình.

Nút Chắn Lọc ID -- ID Mask Node Nút Chắn Lọc ID (lọc chỉ số) có thể được sử dụng để truy cập màn chắn alpha trên mỗi một vật thể hoặc trên mỗi một vật liệu.
Đầu Vào -- Inputs Giá trị ID -- ID value Chỉ số lượt kết xuất cung cấp, Chỉ Số Vật Thể (Object Index) hoặc lượt Chỉ Số Chất Liệu (Material Index), tức là đầu ra của Nút Tầng Lớp Kết Xuất -- Render Layers node hoặc Nút hình ảnh -- Image node với định dạng đa tầng lớp (multi-layer).
Tính Chất -- Properties Chỉ Số -- Index Sự lựa chọn về chỉ số đã xác định trước đây. Chống Răng Cưa -- Anti-Aliased Chức năng hậu xử lý này tinh chỉnh màn chắn. Xin xem thêm mục Chống Răng Cưa -- anti-aliasing .
Đầu Ra -- Outputs Alpha Màn chắn là màu trắng, trong khi vật thể, hoặc những nơi không phải là màn chắn, là màu đen. Nếu vật thể trong suốt thì màn chắn alpha biểu thị điều này dùng các giá trị màu xám. Trong Blender Nội Tại ( Blender Internal ) nếu một vật thể trong suốt nằm ở phía trước một vật thể khác thì màn chắn sẽ không phản ảnh khả năng hiển thị một phần của vật thể nằm ở phía sau.
Sắp Đặt -- Setup An index can be specify for any object or material in the scene. The Object Index can be set in Properties Editor Object ‣ Relations ‣ Pass Index and Material ‣ Settings ‣ Pass Index for the Material Index. To be accessible after rendering, Object Index or Material Index render pass has to be enabled.
Ví Dụ -- Example Trong ví dụ này, hình lập thể màu đỏ phía sau bên trái được gán Chỉ Số Lượt (Pass Index) là 1 và khối lập phương bên phải Chỉ Số Lượt là 2. Nơi hai hình khối giao nhau, sẽ có nhiễu điểm ảnh (pixelation) đáng chú ý xảy ra, vì chúng kết hợp với nhau ở một góc nhọn và có màu khác nhau. Sử dụng màn chắn từ vật thể 1, tức cái đã được làm mịn (chống răng cưa) ở các cạnh, chúng tôi dùng một nút hòa trộn (Mix Node), và đặt phép Nhân (Multiply) để nhân các cạnh đã làm mịn của hình ảnh, hòng loại bỏ các đường nét khó chịu đó đi, và do đó làm mịn được.

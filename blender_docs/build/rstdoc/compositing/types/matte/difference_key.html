Nút Khóa Chênh Lệch -- Difference Key Node This node produces a matte that isolates foreground content by comparing it with a reference background image.
Đầu Vào -- Inputs Hình Ảnh -- Image Contains foreground content against the background that is to be removed. Hình Ảnh -- Image The reference background image.
Tính Chất -- Properties Sai Số -- Tolerance Where pixels match the reference background to within the specified threshold, the matte is made transparent. Suy Giảm Dần -- Falloff Increase to make nearby pixels partially transparent producing a smoother blend along the edges.
Đầu Ra -- Outputs Hình Ảnh -- Image Hình ảnh với kênh alpha của nó được điều chỉnh cho vùng khóa hóa lựa chọn. Lớp Lồng -- Matte Một bản màn chắn alpha trắng-đen của khóa.

Nút Hình Ảnh -- Image Node Nút Hình Ảnh cung cấp bất kỳ hình ảnh nào trong định dạng mà Blender hỗ trợ -- format that is supported by Blender .
Đầu Vào -- Inputs Nút này không có ổ cắm đầu vào.
Tính Chất -- Properties Hình Ảnh -- Image Lựa chọn các loại phương tiện truyền thông khác nhau. Để biết thêm về các điều khiển, xin xem mục Trình Đơn Khối Dữ Liệu -- Data-Block Menu . Để biết thêm về các tùy chọn, xin xem mục Sắp Đặt về Hình Ảnh -- Image Settings . Còn nhiều tùy chọn khác mà mình có thể đặt ở trong vùng thanh bên ( Sidebar region ) nữa.
Đầu Ra -- Outputs Hai ổ cắm đầu tiên là tối thiểu. Hình Ảnh -- Image Đầu ra tiêu chuẩn của hình ảnh. Alpha Giá trị Alpha riêng biệt. Z Tầng lớp chiều sâu Z. Định dạng đa tầng lớp Khi định dạng tập tin đa tầng lớp, như EXR , được nạp vào, mỗi tầng lớp sẽ được biến thành một ổ cắm.

Nút Điểm Ảnh Hóa -- Pixelate Node Cho thêm nút này vào đằng trước một nút tỷ lệ -- scale node để đạt được một hình ảnh điểm ảnh hóa (khối lởm chởm, không mịn màng) từ hình ảnh kết quả được phóng to lên.
Đầu Vào -- Inputs Màu Sắc -- Color Đầu vào hình ảnh tiêu chuẩn.
Tính Chất -- Properties Nút này không có tính chất nào cả.
Đầu Ra -- Outputs Màu Sắc -- Color Đầu ra tiêu chuẩn của hình ảnh.
Ví Dụ -- Example In the compositor, check the Use Nodes checkbox and add an input Image node and an output Viewer node. Connect the Input node to the viewer node and check the Backdrop checkbox in the header. Open an image you would like to pixelate using the open button on the image node. This image should now appear in the backdrop. Now add two scale nodes between the input and output Add ‣ Distort ‣ Scale . Change the values of X and Y to 0.2 in the first scale box and to 5 in the second. The background image will be unchanged. Bây giờ cho thêm một nút Điểm Ảnh Hóa (Pixelate node) giữa hai nút tỷ lệ. Bạn có thể sử dụng Alt-V và V để thu vào, phóng ra hình ảnh nền.

Nút Dịch Chuyển -- Translate Node The translate node moves an image. Could also be used to add a 2D Camera shake.
Đầu Vào -- Inputs Hình Ảnh -- Image Đầu vào hình ảnh tiêu chuẩn. X, Y Used to move the input image horizontally and vertically.
Tính Chất -- Properties Tương Đối -- Relative Percentage translation values relative to the input image size. Tuần Hoàn -- Wrapping Repeat pixels on the other side when they extend over the image dimensions, making endless translating possible. None, X Axis, Y Axis, Both Axis
Đầu Ra -- Outputs Hình Ảnh -- Image Đầu ra tiêu chuẩn của hình ảnh.

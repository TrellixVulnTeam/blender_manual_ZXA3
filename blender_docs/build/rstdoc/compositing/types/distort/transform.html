Nút Biến Hóa -- Transform Node This node combines the functionality of three other nodes: Scale , translate , and rotate nodes.
Đầu Vào -- Inputs Hình Ảnh -- Image Đầu vào hình ảnh tiêu chuẩn. X, Y Used to move the input image horizontally and vertically. Góc Độ -- Angle Used to rotate an image around its center. Positive values rotate counter-clockwise and negative ones clockwise. Tỷ Lệ -- Scale Used to resize the image. The scaling is relative, meaning a value of 0.5 gives half the size and a value of 2.0 gives twice the size of the original image.
Tính Chất -- Properties Bộ/Thanh Lọc -- Filter Interpolation Methods. Gần Nhất -- Nearest Không có nội suy, sử dụng điểm ảnh lân cận nhất. Song Tuyến Tính -- Bilinear Nội suy đơn giản giữa các điểm ảnh lân cận. Song Lập Phương -- Bicubic Nội suy chất lượng cao nhất.
Đầu Ra -- Outputs Hình Ảnh -- Image Đầu ra tiêu chuẩn của hình ảnh.

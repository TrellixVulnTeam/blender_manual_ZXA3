Nút Đính Ghim Góc -- Corner Pin Node The Corner Pin node uses explicit corner values for a plane warp transformation. It works like the Plane Track Deform node, but without using "plane track" data from the Movie Clip Editor.
Đầu Vào -- Inputs Hình Ảnh -- Image Đầu vào hình ảnh tiêu chuẩn. Góc Cạnh -- Corners Four vector inputs to define the plane warping. (Z component of vector inputs is ignored.)
Tính Chất -- Properties Nút này không có tính chất nào cả.
Đầu Ra -- Outputs Hình Ảnh -- Image Standard image output. (The image after distorting.) Mặt Phẳng -- Plane A black-and-white alpha mask of the plane.
Ví Dụ -- Example In the example above, the image of the bird is distorted by the vectors specified by the Corner Pin node.

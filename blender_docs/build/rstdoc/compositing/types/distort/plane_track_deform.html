Nút Biến Dạng Giám Sát Mặt Phẳng -- Plane Track Deform Node The Plane Track Deform Node is used to incorporate the special "plane track" in your composite by checking areas which are planes, and replacing their footage with some other image.
Giám Sát Mặt Phẳng -- Plane Track Before using this node, plane track for the footage should be made in the Movie Clip Editor .
Đầu Vào -- Inputs Hình Ảnh -- Image Image to put in place of the plane track, and thus, override that area in the movie clip.
Tính Chất -- Properties Đoạn Phim -- Movie Clip Used to select the movie clip whose plane track to use. For controls see Trình Đơn Khối Dữ Liệu -- Data-Block Menu . Vật Thể -- Object Used to select the object to which the plane track is linked. Giám Sát/Rãnh -- Track Used to select the plane track to use. Làm Nhòe Chuyển Động -- Motion Blur Specify whether to use blur caused by motion of the plane track or not. Lượng Mẫu Vật -- Samples Set the number of samples to take for each frame. The higher this number, the smoother the blur effect, but the longer the render, as each virtual intermediate frame has to be rendered. Samples are taken only from the next frame, not the previous one. Therefore the blurred object will appear to be slightly ahead of how it would look without motion blur. Màn Trập -- Shutter Time (in frames) the shutter is open. If you are rendering at 24 fps, and the Shutter is set to 0.5, the time in between frames is 41.67 ms, so the shutter is open for half that, 20.83 ms.
Đầu Ra -- Outputs Hình Ảnh -- Image The output by perspective wrapping the image to that plane track. Mặt Phẳng -- Plane Produces a black-and-white mask of the plane track.

Nút Xoay Chiều -- Rotate Node This node rotates an image.
Đầu Vào -- Inputs Hình Ảnh -- Image Đầu vào hình ảnh tiêu chuẩn. Góc Độ -- Degr Rotation angle in degree. Positive values rotate clockwise and negative ones counterclockwise.
Tính Chất -- Properties Bộ/Thanh Lọc -- Filter Interpolation Methods. Gần Nhất -- Nearest Không có nội suy, sử dụng điểm ảnh lân cận nhất. Song Tuyến Tính -- Bilinear Nội suy đơn giản giữa các điểm ảnh lân cận. Song Lập Phương -- Bicubic Nội suy chất lượng cao nhất.
Đầu Ra -- Outputs Hình Ảnh -- Image Đầu ra tiêu chuẩn của hình ảnh.

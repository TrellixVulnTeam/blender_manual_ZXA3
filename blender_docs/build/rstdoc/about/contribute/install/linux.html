Cài Đặt các Phụ Thuộc -- Installing Dependencies Dưới đây là danh sách các lệnh cài đặt cho các phiên bản phân phối Linux phổ biến. Đối với hệ thống thích hợp, hãy thi hành lệnh sau trong thiết bị đầu cuối (bàn giao tiếp): Debian/Ubuntu sudo apt-get install python3 python3-pip subversion Redhat/Fedora sudo yum install python python-pip Arch Linux sudo pacman -S python python-pip subversion
Tải từ kho lưu trữ xuống máy -- Downloading the Repository Chỉ cần kiểm xuất (checkout) kho lưu trữ của bản hướng dẫn sử dụng Blender bằng cách sử dụng cd ~ svn checkout https://svn.blender.org/svnroot/bf-manual/trunk/blender_docs Kho lưu trữ bây giờ sẽ được tải xuống và mình có thể phải chờ mất vài phút, tùy theo chất lượng đường kết nối Internet của bạn.
Thiết Lập Môi Trường Biên Dịch -- Setting up the Build Environment Mở một cửa sổ Thiết Bị Đầu Cuối (Bàn Giao Tiếp để đánh dòng lệnh). Bước vào thư mục blender_docs vừa được kiến tạo và bổ sung bởi việc kiểm xuất (checkout) SVN: cd ~/blender_docs Inside that folder is a file called requirements.txt which contains a list of all the dependencies we need. To install these dependencies, we can use the pip3 command: sudo pip3 install -r requirements.txt Every now and then you may want to make sure your dependencies are up to date using: sudo pip3 install -r requirements.txt --upgrade Tiếp tục với bước tiếp theo: Biên Tập .

Text Strips The Text strip allows you to directly display text in the Sequence editor. The strip will display the text inserted in its text field on the final sequence. All Text strips in a video sequence can be exported as a SubRip file. This is useful when using Text Strips as subtitles.
Tùy Chọn -- Options Văn Bản -- Text The actual text displayed.
Style Font Trình Đơn Khối Dữ Liệu -- Data-Block Menu to choose which font-file is used to render the text. Kích Thước -- Size Size of the text. Màu Sắc -- Color The text color. Ngả Bóng Tối -- Shadow Creates a shadow under the text.
Layout Location X/Y Positions the text on the X, Y axis. Alignment X/Y Horizontal (X) or vertical (Y) alignment of the text relative to the location. Chiều Rộng Xuống Dòng -- Wrap Width Wraps the text by the percentage of the frame width, setting this to zero disables word wrapping.

Lựa Chọn -- Selecting Box Select Click and drag to box select multiple nodes. Alternatively, B starts the bounding box selection process as well. Chọn bằng Dây Thòng Lọng -- Lasso Select Ctrl-Alt-LMB click and drag starts a lasso selection. (Hủy) Chọn Toàn Bộ -- (De)select All A Đảo Nghịch -- Inverse Ctrl-I Chọn cái Kết Nối Từ -- Select Linked From L Chọn cái Kết Nối Tới -- Select Linked To Shift-L Chọn Theo Nhóm Shift-G -- Select Grouped Shift-G Selects similar nodes to the active node by their properties . Thể Loại -- Type The node type. e.g. all Math nodes. Màu Sắc -- Color The color property. Tiền Tố, Hậu Tố -- Prefix, Suffix Matches the name property from start/end of the text. Kích hoạt chức năng nhảy đến Trước Đây/Tiếp Tới nếu Cùng Thể Loại -- Activate Same Type Previous/Next Shift-] / Shift-[ Tìm nút trước đây/tiếp tới có cùng thể loại, kích hoạt nút, và đảm bảo là nút ấy nhìn thấy được. Lùng Tìm Nút Ctrl-F -- Find Node Ctrl-F To search for a node. On selecting a node, it activates the node and makes sure the node is visible. Chọn nhiều -- Select multiple Dùng Shift-LMB hoặc Shift-RMB để chọn nhiều nốt.

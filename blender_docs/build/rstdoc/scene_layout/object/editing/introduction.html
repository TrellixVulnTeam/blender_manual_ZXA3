Giới Thiệu -- Introduction This section describes tools for manipulating objects in Object Mode. All editing tools can be found in Object menu.
Hội Nhập/Chắp Nối -- Join Tham Chiếu -- Reference Chế Độ -- Mode Chế Độ Vật Thể -- Object Mode Trình Đơn -- Menu Vật Thể (Object) ‣ Chắp Nối (Join) Phím Tắt -- Hotkey Ctrl-J Join merges all selected objects into the last selected Active object. All object data is linked to the active object (which must be selected). All objects must be of the same type: mesh, curve, surface or armature. If several curves are joined, each one will keep its subtype (NURBS or Bézier). Object data has many attributes which may be handled when joining. Materials, vertex groups, UV and Vertex layers will be merged. Modifiers, constraints, groups and parent relationships are ignored when joining and will not be applied to the active object.
Chuyển Đổi Sang -- Convert To
Đường Cong từ Khung Lưới/Văn Bản -- Curve from Mesh/Text Tham Chiếu -- Reference Chế Độ -- Mode Chế Độ Vật Thể -- Object Mode Trình Đơn -- Menu Vật Thể (Object) ‣ Chuyển Đổi thành (Convert to) ‣ Đường Cong từ Khung Lưới/Văn Bản (Curve from Mesh/Text) Mesh objects and text objects can be converted into curve objects. In mesh objects, only edges belonging to no faces will be taken into account. The resulting curve will be a Poly curve type, but can be converted to have smooth segments as described in Set Spline Type .
Khung Lưới từ Đường Cong/Siêu Cầu/Bề Mặt/Văn Bản -- Mesh from Curve/Metaball/Surface/Text Tham Chiếu -- Reference Chế Độ -- Mode Chế Độ Vật Thể -- Object Mode Trình Đơn -- Menu Object ‣ Convert to ‣ Mesh from Curve/Meta/Surf/Text Converts the selected curve, metaball, surface and text objects to mesh objects. The actual defined resolution of these objects will be taken into account for the conversion. Note that it also keeps the faces and volumes created by closed and extruded curves.
Tùy Chọn -- Options Duy Trì Bản Gốc -- Keep Original Duplicates the original object before converting it.
Hiện/Ẩn -- Show/Hide Tham Chiếu -- Reference Chế Độ -- Mode Toàn Bộ các Chế Độ -- All Modes Trình Đơn -- Menu Vật Thể (Object) ‣ Hiện/Ẩn (Show/Hide) Show Hidden Objects Alt-H Reveals all hidden objects. Ẩn Giấu cái Được Chọn -- Hide Selected H Hides all selected objects. Ẩn Giấu Cái Không Được Chọn -- Hide Unselected Shift-H Hides all unselected objects of the scene.
Xóa -- Delete Tham Chiếu -- Reference Chế Độ -- Mode Chế Độ Vật Thể -- Object Mode Trình Đơn -- Menu Vật Thể (Object) ‣ Xóa (Delete) Phím Tắt -- Hotkey X or Delete The selected objects are deleted from the scene. Delete Globally Delete the selected objects from all scenes.

Brush Brush The brush presets data-block menu. Color The color of the brush. See Trình Chọn Màu -- Color Picker . Press S on any part of the image to sample that color and set it as the brush color. Hold Ctrl while painting to temporally paint with the secondary color. Flip (cycle icon) X Swaps the primary and secondary colors. Note that Vertex Paint works in sRGB space , and the RGB representation of the same colors will be different between the paint tools and the materials that are in linear space. Radius Set the radius of the brush. Strength Set the strength of the brush's effect. Blend See Color Blend Modes .
Options Accumulate This will allow a stroke to accumulate on itself, just like an airbrush would do. Affect Alpha When this is disabled, it locks (prevents changes) the alpha channel while painting. Front Faces Only Only paint on the front side of faces. 2D Falloff This turns the brush influence into a cylinder (the depth along the view is ignored) instead of a sphere.

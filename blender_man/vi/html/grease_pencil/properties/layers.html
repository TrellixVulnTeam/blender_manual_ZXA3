Các Tầng Lớp

Tham Chiếu

Chế Độ

Tổng Thể

Bảng

Chọn Lựa (Select) ‣ Nới Rộng/Thu Nhỏ (More/Less)

Tổ Hợp Phím Tắt

Y

Danh Sách Tầng Lớp

Grease Pencil objects each have a list of 2D layers for grouping and arranging strokes in a List view. Any stroke can only belong to a single 2D layer. There is always only one active layer in the list (the selected one). When you draw, the new strokes are added to the active layer. By default the view order of the layers in the viewport is top to bottom.

Every layer correspond to a channel in the Dope Sheet editor (in Grease Pencil mode). See Dope Sheet for more information.

Layers can also be used together with Modifiers to only affects part of your drawing. See Modifiers for more information.

Mẹo

Sometimes the layers you are not working on can be a distraction. Activate Fade Layers in overlays to control the opacity of the non-active layers. See Overlays for more information.

Next to the layer name there are four icons buttons that control common properties of the layer:

Dùng Màn Chắn (biểu tượng màn chắn)

Toggle the affect of Masks on the layer.

Bóc Vỏ Hành (biểu tượng bóc vỏ hành)

Toggle using the layer for Onion Skinning.

Đã được/bị ẩn giấu đi (biểu tượng con mắt)

Toggle layer visibility in the viewport and in render.

Khóa (biểu tượng cái khóa)

Toggle layer from being editable.

Những Đặc Biệt về Tầng Lớp

Operators for working with layers.

Sao Chép Tầng Lớp

Makes an exact copy of the selected layer appending a number to differentiate its name.

Sao Chép các Khung Khóa Trống

Makes a copy of the selected layer but with empty keyframes. Useful to easily have empty keyframes preset to work on the cleanup or filling process.

Hiển Thị Toàn Bộ

Bật/Tắt mặt phẳng nền trong góc nhìn Thực Tế Ảo.

Ẩn Giấu những cái Khác

Turns off the visibility of every layer in the list except the active one.

Khóa Toàn Bộ

Locks edition of all the layers in the list.

Mở Khóa Toàn Bộ

Unlocks edition of all the layers in the list.

Tự Động Khóa Tầng Bất Hoạt

Locks automatically the edition of every layer in the list except the active one. This way you avoid to make unwanted changes in other layers without the need to lock them every time.

Cấm Biên Soạn các Nguyên Vật Liệu đã Khóa

Avoids editing locked materials in the layer. When disabled, any material can be edited even if they are locked in the material list.

Thêm :kbd:`Shift-Ctrl- = `

Merge the selected layer with the layer below, the new layer keeps the name of the lower layer.

Sao Chép Tầng Lớp sang cái Đã Chọn

Tạo một mảng các đường cong đã chọn trong Chế Độ Biên Soạn.

Sao Chép Toàn Bộ các Tầng Lớp sang cái Đã Chọn

Tạo một mảng các đường cong đã chọn trong Chế Độ Biên Soạn.

Ẩn-Hiện (biểu tượng màn hình)

Toggle whether the active layer is the only one that can be edited and is visible.

Khóa (biểu tượng cái khóa)

Toggle whether the active layer is the only one that can be edited.

Below the layers list there are additional common settings:

Pha/Hòa Trộn/chuyển đổi/chuyển đổi

The layer blending operation to perform. See Color Blend Modes.

Độ Đục

Used to set the opacity of the layer.

Sử Dụng Ánh Sáng

When enabled, the layer is affected by lights.

Màn Chắn

In a List view of layers affected by a layer mask. See Masks for more information.

Biến Hóa

Allows per-layer location, rotation and scale transformations.

Điều Chỉnh

Màu Nhuốm

Color that tint any material colors used in the layer.

Hệ Số

Controls the amount of tint color to apply.

Chiều Rộng Nét Vẽ

Thickness value that override the strokes thickness in the layer.

Mối Quan Hệ

Phụ Huynh/Kiểu/Thể Loại

Select a Parent object and Type to manipulate the layer. The layer will inherit the transformations of the parent, this is especially useful when rigging for cut-out animation.

Chỉ số của lượt

The layer index number can be used with some modifiers to restrict changes to only certain areas.

See Modifiers for more information.

Tầng góc nhìn

Defines the View Layer to use for the Grease Pencil layer. If empty, the layer will be included in all View Layers. This is useful to separate drawings parts for compositing.

Đã Bị Giải Hoạt (Tắt) (màu đỏ) -- Disabled (red)

If enabled no masks on the layer are included in the view layer render.

Hiển Thị

Màu Kênh Tùy Chọn

Sets the color to use in the channel region of the Dope Sheet.

Duy Hiển Thị tầng lớp đã Khung Khóa Hóa

Makes the layer visible in the viewport only if it has a keyframe in the actual frame. This helps for example when you are in the inking process using the Fill tool and want to only see the strokes that are in the actual frame to avoid fill in unwanted regions.

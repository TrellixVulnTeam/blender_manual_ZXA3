Tóc -- Hair

Hair type particle system can be used for strand-like objects, such as hair, fur, grass, quills, etc.

Tính Chất -- Properties

Thuộc Tính -- Attributes

The Attributes panel contains different hair characteristics such as the position and color of hair strands.

Use the List View to manage attributes.

Attribute Types

Vị Trí -- Position

The position of the point in 3D space.

Thể Loại -- Type

Vectơ -- Vector

Bán Kính -- Radius

The radius of each point.

Thể Loại -- Type

Float

Color

The color of each point.

Thể Loại -- Type

Color

Custom Attributes

Custom attribute can be given to particles to hold a custom characteristic.

Tên -- Name

The name of the attribute.

Data Type

The type of data to store in the attribute.

Float

Floating-point value

Integer

32-bit integer

Vectơ -- Vector

3D vector with floating-point values

Color

RGBA color with floating-point precision

Byte Color

RGBA color with 8-bit precision

String

Text string

Domain

The type of element the attribute is stored in.

Điểm -- Point

The attribute is stored per hair point.

Curve -- Đường Cong

The attribute is stored per hair curve.

Tính Chất Tùy Chỉnh -- Custom Properties

Xin xem thêm trang Tính Chất Tự Tạo -- Custom Properties để biết thêm thông tin.

Vòng Mạch Phân Chia

Tham Chiếu

Chế Độ

Chế Độ Biên Soạn

Công Cụ

Thanh Bên (Sidebar) ‣ Thẻ Biên Soạn (Edit tab)

Tổ Hợp Phím Tắt

Ctrl-R

The Loop Cut tool is a modal tool version of the Phân Chia Vòng Mạch và Trượt Đẩy operator. This tool splits a loop of faces by inserting new edge loops intersecting the chosen edge.

Sử Dụng

The tool is interactive and has two steps:

  1. Pre-Visualizing the Cut

    After the tool is activated, move the cursor over a desired edge. The cut to be made is marked with a magenta colored line as you move the mouse over the various edges. The to be created edge loop stops at the poles (tris and n-gons) where the existing face loop terminates.

  2. Biến dạng chúng -- Deforming them

    Once the desired location of the new edge loop is found, the edge loop can be created via LMB.

Sắp Đặt Công Cụ

Số phân đoạn

Increases and decreases the number of cuts to create. These cuts are uniformly distributed in the original face loop, and you will not be able to control their positions.

Chỉnh Sửa UV

Corrects the corresponding UV coordinates, if these exist, to avoid image distortions.

Tùy Chọn

After the modal tool is run the Loop Cut and Slide Options are available in the Điều Chỉnh Thao Tác Trước Đây panel.

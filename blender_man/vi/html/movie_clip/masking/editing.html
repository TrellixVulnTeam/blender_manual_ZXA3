Biên Soạn

The tools and panels available to edit masks are the same in both editors. Editing of mask splines happens in a way similar to editing Bézier curves or paths in GIMP or other curve editors.

Mẹo

To get interactive feedback on the resulting mask, a Mask node can be connected directly to a Viewer node in the Compositor, which will then keep updating the compositing result while editing.

Biến Hóa

Tham Chiếu

Chế Độ

Chế Độ Màn Chắn

Trình Đơn

Xương (Bone) ‣ Biến Hóa (Transform)

Di Chuyển (Move) G

Change the location of control points. Control points can also be moved with LMB. The whole spline can be moved by dragging the center dot with LMB.

Xoay Chiều (Rotate) R

Change the location of control points by rotating about a pivot point.

Đổi Tỷ Lệ (Scale) S

Change the location of control points by expanding the distance between points.

Thành Hình Cầu (To Sphere) Shift-Alt-S

Morphs the control points to the shape of a circle.

Xô Nghiêng (Shear) Shift-Ctrl-Alt-S

Shifts control points along a defined axis so parallel control points move past one another.

Đẩn/Kéo (Push/Pull)

Moves the control points closer together (Push) or further apart (Pull).

Đổi Tỷ Lệ Mép Nhòe Alt-S

Will scale the feather size.

Hoạt Họa

Tham Chiếu

Chế Độ

Chế Độ Màn Chắn

Trình Đơn

Xương (Bone) ‣ Mối Quan Hệ (Relations)

Masks can be animated with the shape keying system. This can be useful when there are not enough good feature points to track in the footage, or the mask is not based on footage. Mask animation timing can be edited from the Dope Sheet's Mask Mode.

Chèn Thêm Hình Dạng Khóa I

Will insert a shape key for the active mask layer at the current frame. This works on the level of mask layers, so inserting a shape key will keyframe all the splines and points contained in it.

Xóa Hình Dạng Khóa Alt-I

Will clear the shape key for the active mask layer at the current frame.

Hoàn Lại Mép Nhòe của Hoạt Hình

Resets the feather offset across all animated frames.

Khóa Hóa Lại những Điểm của các Hình Dạng được Chọn

Re-interpolate selected points on across the range of keys selected in the Dope Sheet.

Linh Tinh/Những cái Khác

Trượt Đẩy Theo Đường Cong của Spline NCT (LMB)

Moves the curve and/or control points by clicking on them and dragging.

Thêm điểm đỉnh và Trượt Đẩy Ctrl-NCT (LMB)

Inserts new control points and defines handle orientations by a continued mouse drag. If the last point was selected, double-click will also close the curve.

Thêm điểm đỉnh cho Mép Nhòe và Trượt Đẩy Shift-Ctrl-NCT (LMB)

Inserts new feather control points that can be transformed independently of the main spline curve. If no feather mask is in use this will create a basic feather mask to the curve.

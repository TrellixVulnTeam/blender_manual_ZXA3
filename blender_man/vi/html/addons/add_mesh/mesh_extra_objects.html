Thêm Vật Thể Khung Lưới Bổ Sung

This add-on groups many mesh object creation add-ons into a single one.

Kích Hoạt -- Activation

  • Mở Blender và chuyển đến Cấu Hình, sau đó đến thẻ Trình Bổ Sung.

  • Click Add Mesh then Extra Objects to enable the script.

Giao Diện -- Interface

Located in the 3D Viewport ‣ Add ‣ Mesh menu.

Thông Tin -- Information

Included mesh object add-ons:

Phụ Huynh Hóa vào Vật Thể Trống Rỗng bởi Liero

Parent the selected mesh to an empty object type.

Bề Mặt Hàm 3D bởi Buerbaum Martin (Pontiac), Elod Csirmaz

Create objects with XYZ math functions. Includes several presets by elfnor.

Beam Builder bởi revolt_randy, Jambay

Create five types of beams.

Bánh Răng Cưa bởi Michel J. Anders (varkenvarken)

Tạo bánh xe răng cưa và trục vít truyền động.

Đá Quý bởi Pontiac, Fourmadmen, Dreampainter, Dominic Kroper, (dommetysk)

Create three types of diamond shapes.

Tàng ong bởi Kayo Phoenix

Create a honeycomb patterned mesh.

Bọt Biển Menger bởi sugiany

Create iterated Menger sponges.

Khớp Nối Đường Ống bởi Buerbaum Martin (Pontiac)

Create angled and crossed pipes.

Kim Tự Tháp Bậc Thang bởi Phil Cote (cotejrp1)

Create a step pyramid with variable settings.

Hình Lập Phương bo Tròn bởi Alain Ducharme (phymec)

Create a quad mesh sphere with several presets.

Khối Lập Thể Thông Thường bởi DreamPainter

Create polyhedron based objects.

Hình Ngôi Sao bởi Fourmadmen

Create a simple star shape.

Hình Xuyến Siêu Cấp bởi DreamPainter

Create a torus object with new parameters for boxed shape and more.

Ấm trà của Anthony D'Agostino

Traditional style mesh teapot and a secondary spoon object.

Nút Thắt Hình Xuyến bởi Anthony D'Agostino

A simple mesh torus knot with three types.

Hình Tam Giác bởi Sjaak-de-Draak

Create math based triangles.

Hình Xuyến Xoắn bởi Paulo_Gomes

Standard torus object with a variable twisted mesh.

Thêm Điểm Đỉnh bởi meta-androcto, Pablo Vazquez, Liero, Richard Wilks

Add a single vertex object or object origin as a fast start to mesh editing.

Xưởng Sản Xuất Tường Nhà bởi dudecon, jambay

Add castle type walls with settings for openings.

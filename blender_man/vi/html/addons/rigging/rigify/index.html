Dựng Giàn Điều Khiển

Miêu Tả

Automatic rigging from building-block components.

Vị Trí

Properties ‣ Armature, Bone, 3D Viewport ‣ Tools panel, 3D Viewport ‣ Add menu ‣ Armature

Tập Tin

rigify folder

Tác Giả

Nathan Vegdahl, Lucio Rossi, Ivan Cappiello, Alexander Gavrilov

Giấy Phép

GPL

Ghi Chú

Trình bổ sung này được gói cùng với Blender.

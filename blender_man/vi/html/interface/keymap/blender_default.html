Cấu hình về bố trí phím

Chọn bằng Nút Chuột

Controls which mouse button, either right or left, is used to select items in Blender. If Left is selected the RMB will be a context sensitive menu, if Right is selected the LMB will place the 3D Cursor.

Hành động của phím Dấu Cách

Sắp đặt hành vi của Dấu Cách (Spacebar). Những bố trí này, và cả những sắp đặt đường cắt ngắn khác, có thể được sửa đổi trong Cấu Hình Về Bố Trí Phím (Keymap Preferences).

Chơi

Starts playing through the Timeline, this option is good for animation or video editing work.

Công Cụ

Mở Thanh Công Cụ nằm bên dưới con trỏ để nhanh tróng chuyển đổi công cụ đang hoạt động. Tùy chọn này là một tùy chọn rất tốt nếu chúng ta làm việc nhiều về kiến tạo mô hình hoặc dàn dựng.

Tìm Kiếm

Opens up the Menu Search. This option is good for someone who is new to Blender and is unfamiliar with the menus and shortcuts.

Kích Hoạt Sự Kiện Gizmo

The activation event for gizmos that support drag motion. This option is only available when Left click Select with Mouse Button is chosen.

Bấm

Allows immediate activation, preventing click events being passed to the tool.

Kéo Rê

Allows click events to pass through to the tool, adding a small delay.

Lựa Chọn Bằng Chuột -- Mouse selection

The default action for the right mouse button. This option is only available when Right click Select with Mouse Button is chosen.

Chọn lựa hoặc di chuyển

Right mouse always tweaks the selected item.

Duy Lựa Chọn -- Selection Only

Right mouse uses the selection tool.

Công Cụ -- Tools

The method of keys to activate tools such as move, rotate, and scale.

Ngay lập tức/trực tiếp/gần gũi nhất/sát cạnh

Activate actions immediately.

Công Cụ Đang Hoạt Động

Activate the tool for editors that support tools.

Bấm Alt Nhấp Chuột với Công Cụ Dấu Nhắc

Tapping Alt shows a prompt in the status bar prompting a second keystroke to activate the tool. Note this option is not available when using Emulate 3 Button Mouse.

Canh Hàng Công Cụ

Hold Alt to use the Active Tool when the gizmo would normally be required. This option is only available when Left click Select with Mouse Button is chosen. Note this option is not available when using Emulate 3 Button Mouse.

Alt Cursor Access

Hold Alt-LMB to place the Cursor (instead of LMB), allows tools to activate on press instead of drag. This option is only available when Right click Select with Mouse Button is chosen. Note this option is not available when using Emulate 3 Button Mouse.

Bật/Tắt Chọn Toàn Bộ

Causes selection shortcut A to deselect all when any selection exists.

Cổng Nhìn 3D

Hành Động Dấu Huyền / Dấu Ngã
Điều Hướng

Navigation pie menu, useful on systems without a numeric keypad.

Gizmos

Transform gizmos pie menu, useful for quickly switching between transform gizmos.

Hành Động của Chuột Giữa

The action when MMB dragging in the viewport, this also applies to trackpads.

Quỹ Đạo

Rotates the view around a pivot point, Shift-MMB is used for panning the view.

Lia

Shifts the view towards the mouse, Shift-MMB is used for orbiting the view.

Hành Động Kéo Rê Chuột Giữa Trong Khi Bấm Alt
Tương Đối

Set the view axis where each mouse direction maps to an axis relative to the current orientation.

Tuyệt Đối

Set the view axis where each mouse direction always maps to the same axis.

Bấm phím Tab để xem Trình Đơn Rẻ Quạt

Causes Tab to open a pie menu (swaps Tab and Ctrl-Tab).

Trình Đơn Hình Rẻ Quạt khi Kéo Rê

This allows keys to have a secondary drag action.

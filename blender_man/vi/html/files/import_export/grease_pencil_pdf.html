Định Dạng Tài Liệu Di Động (PDF) thành Bút Chì Dầu

This format is use for interchanging PDFs between applications, it support the export of Grease Pencil animation creating one page in the PDF document for each keyframe selected.

Cảnh báo

The exporter only works in Object Mode.

Các Tùy Chọn về Xuất Khẩu

The following options are available when exporting to PDF:

Vật Thể

Determine which objects will be included in the export.

Đang/Hoạt Động

Export only the active Grease Pencil object.

Cái được Chọn

Chỉ xuất các vật thể đã chọn mà thôi.

Hữu Hình

Export all visible Grease Pencil object in the scene.

Khung Hình

Determine which frames will be included in the export.

Đang/Hoạt Động

Đặt giá trị cho khung khóa đang hoạt động.

Cái được Chọn

Export all selected keyframes as different PDF pages.

Ghi chú

To enable multi-keyframe selection you must enable Multiframe Edition. See Multiframe Edition for more information.

Lấy Mẫu Vật

Precision for the stroke sampling. Low values mean a more accurate result.

Phủ/Tô Kín/Đổ Sơn

When enabled, Export the Grease Pencil strokes fill.

Bình Thường Hóa

When enabled, Export strokes with constant thickness.

Ghi chú

The export of the Grease Pencil strokes is doing always from camera view.

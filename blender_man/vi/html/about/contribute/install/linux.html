Cài Đặt các Phụ Thuộc -- Installing Dependencies

Dưới đây là danh sách các lệnh cài đặt cho các phiên bản phân phối Linux phổ biến.

Đối với hệ thống thích hợp, hãy thi hành lệnh sau trong thiết bị đầu cuối:

Debian/Ubuntu:

sudo apt-get install python3 python3-pip subversion

Redhat/Fedora:

sudo yum install python python-pip

Arch Linux:

sudo pacman -S python python-pip subversion

Cảnh báo

Python 3.10 là phiên bản chưa được hỗ trợ. Chúng tôi đề cử là các bạn nên sử dụng phiên bản Python 3.9 hoặc trước đó thôi nhé.

Tải Kho Lưu Trữ về Máy -- Downloading the Repository

Chỉ cần kiểm xuất kho lưu trữ của bản hướng dẫn sử dụng Blender bằng cách dùng:

cd ~
svn checkout https://svn.blender.org/svnroot/bf-manual/trunk/blender_docs

Kho lưu trữ lúc này sẽ được tải xuống và mình có thể phải chờ mất vài phút, tùy theo chất lượng đường kết nối Internet của bạn.

Thiết Lập Môi Trường Biên Dịch -- Setting up the Build Environment

  • Mở một Cửa Sổ Thiết Bị Đầu Cuối.

  • Bước vào thư mục blender_docs vừa được kiến tạo và bổ sung bởi việc kiểm xuất SVN:

    cd ~/blender_docs

  • Bên trong thư mục đó có một tập tin với cái tên là requirements.txt, và trong đó có chứa một danh sách của tất cả các phần mềm phụ thuộc mà chúng ta cần phải có. Để cài đặt các phần phụ thuộc này, chúng ta có thể sử dụng lệnh pip3 như sau:

    sudo pip3 install -r requirements.txt

Ghi chú

Thỉnh thoảng, bạn có thể muốn các thư viện phụ thuộc của bạn được cập nhật với những thay đổi gần đây nhất, nếu vậy thì xin hãy sử dụng:

sudo pip3 install -r requirements.txt --upgrade

Thêm/Xóa một Ràng Buộc/cho/cộng thêm vào/Xóa Bớt một Ràng Buộc

What is described on this page about Object Constraints can be also be applied on Bone Constraints.

Thẻ

Tham Chiếu

Chế Độ

Chế Độ Vật Thể

Trình Đơn

Properties ‣ Constraint tab

To add a constraint click on the Add Object Constraint menu in the Constraints tab.

To remove a constraint click on the "X" button in the header.

Trình Đơn

Thêm Ràng Buộc (với Mục Tiêu)

Tham Chiếu

Chế Độ

Chế Độ Vật Thể và Chế Độ Tư Thế

Trình Đơn

Object ‣ Constraint ‣ Add Constraint (with Targets)

Adds a constraint to the active object. The type of constraint must be chosen from a pop-up menu, though it can be changed later from the Add Constraint (with Targets) Điều Chỉnh Thao Tác Trước Đây panel. If there is an other object selected besides the active one, that object will be the constraint target (if the chosen constraint accepts targets).

When using a bone from another armature as the target for a constraint, the tool will look inside the non-active armature and use its active bone, provided that armature is in Pose Mode.

Sao Chép Ràng Buộc sang các Vật Thể được Chọn

Tham Chiếu

Chế Độ

Chế Độ Vật Thể và Chế Độ Tư Thế

Trình Đơn

Object ‣ Constraint ‣ Copy Constraints to Selected Objects

Copies the active object Constraints to the rest of the selected objects.

Xóa các Ràng Buộc của Vật Thể

Tham Chiếu

Chế Độ

Chế Độ Vật Thể và Chế Độ Tư Thế

Bảng

Object ‣ Constraint ‣ Clear Object Constraints

Removes all Constraints of the selected object(s).

Giám Sát/Rãnh

Tham Chiếu

Chế Độ

Chế Độ Vật Thể

Bảng

Object ‣ Track

These tools add a tracking constraint to the selected objects; the target object of the constraint will be the active object, which won't have a constraint added.

  • Damped Track Constraint

  • Track To Constraint

  • Lock Track Constraint

Xóa Giám Sát

Removes all Damped Track, Track To and Lock Track Constraints from the selected objects.

Xóa và Duy Trì Biến Hóa (Xóa Giám Sát)

Removes all Track Constraint from the selected objects, while keeping the final transform caused by them.

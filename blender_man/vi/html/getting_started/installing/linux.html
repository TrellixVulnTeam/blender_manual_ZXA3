Cài đặt trên hệ điều hành Linux

Xin hãy kiểm tra chi tiết trên trang Tải Blender Xuống Máy (Downloading Blender) để biết được các đòi hỏi tối thiểu của phần mềm, và đồng thời, nơi mình có thể lấy được một bản Blender cho cá nhân nữa (nếu bạn chưa làm việc này).

Cài đặt từ blender.org

Tải xuống phiên bản Linux cho kiến trúc (cấu hình hệ điều hành) của bạn và giải nén tập tin ở vị trí mong muốn (ví dụ: ~/software or /usr/local).

Lúc này, Blender có thể được khởi chạy bằng cách nhấp đúp vào tập tin thực thi.

When using this method of installation, it is possible to have multiple versions of Blender installed.

Để truy cập dễ dàng, bạn có thể cấu hình hệ thống của bạn bằng cách cho thêm một trình mục hoặc một đường tắt vào trong Blender. bạn cũng có thể liên các tập tin blend với Blender hầu cho khi chọn lựa trong Trình Duyệt Thảo tập tin (File browser) thì chúng sẽ được tự động mở trong Blender. Những Sắp đặt này thường thấy trong các Sắp đặt của Trình Quản Lý Cửa Sổ (Window Manager) như (KDE, Gnome, Unity)

Cài Đặt từ Trình Quản Lý Gói Phần Mềm

Một số bản phân phối hệ điều hành Linux có thể có một gói Blender đặc trưng riêng cho mình trên các kho của chúng.

Installing Blender via the distribution's native mechanisms ensures consistency with other packages on the system and may provide other features (given by the package manager), such as listing of packages, update notifications and automatic menu configuration. Be aware, though, that the package may be outdated compared to the latest official release, or not include some features of Blender. For example, some distributions do not build Blender with Cycles GPU rendering support, for licensing or other reasons.

Nếu cần có một gói phần mềm cụ thể nào đó cho phiên bản phân phối của mình thì bạn có thể chọn cái nào thích hợp và thuận tiện nhất, nếu không, việc lấy bản nhị phân chính thức trên blender.org là một điều rất nên làm.

Cài Đặt từ Trình Quản Lý Snap

Snap is a universal package manager designed to work across a range of distributions. Assuming snap is already installed, Blender can be installed through snap with:

snap install blender

Installing from this method has a benefit that updates to Blender are automatically installed. Blender from Snap should have a more consistent distribution then individual package managers.

Thi Hành dùng Thiết Bị Đầu Cuối (Bàn Giao Tiếp, dòng lệnh)

See Launching from the terminal.

Tránh Xung Đột trong Bố Trí Phím Alt+Bấm Chuột

Rất nhiều Trình Quản Lý Cửa Sổ sử dụng bố trí phím mặc định :kbd:`Alt-NCT (LMB) để di chuyển các cửa sổ, song đường tắt này là đường tắt mà Blender sử dụng để mô phỏng chuột có ba nút bấm. Bạn có thể hoặc là tắt chức năng này đi, bằng các bấm vào Sở Thích Người Dùng (User Preferences) ‣ Đầu Vào (Input) ‣ Mô Phỏng Chuột 3 Nút (Emulate 3 Button Mouse) hoặc bạn có thể đổi sắp đặt của Trình Quản Lý Cửa Sổ và sử dụng phím Meta thay thế (còn được gọi là phím Quản Lý (Super) hoặc Cửa Sổ (Windows)):

Gnome

Unity/Gnome: Nhập nội dung sau vào dòng lệnh (phải đăng xuất vì lệnh chỉ có hiệu lực tại lần đăng nhập tiếp theo):

gsettings set org.gnome.desktop.wm.preferences mouse-button-modifier '<Super>'
KDE

KDE: Thiết Lập Hệ Thống (System Settings) ‣ Quản Lý Cửa Sổ (Window Management) ‣ Ứng Xử của Cửa Sổ (Window Behavior) ‣ Hành Động Cửa Sổ (Window Actions) Đổi phím 'Alt' sang phím 'Meta' (Ở hộp Phím Bổ Trợ (Modifier key)).

Cập nhật trên Linux

On Linux there are various ways of updating Blender. This section covers the most common approaches.

Cập nhật từ địa điểm blender.org

When an update for Blender is released, it can be downloaded directly from the Blender website and installed using the steps described in the section Install from blender.org.

Cập Nhật bằng bằng một Trình Quản Lý Gói Phần Mềm

Many Linux distributions have packages for Blender available, which can be installed using the distribution's package manager. After installation, Blender can be updated using the same steps as updating any other application.

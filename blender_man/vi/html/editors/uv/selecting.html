Lựa Chọn

Selection tools are available in the Select Menu in the header, and the shortcuts listed below.

Đồng Bộ Hóa Lựa Chọn

Turning on the Sync Selection button causes selection of components in the 3D Viewport to sync with their corresponding elements in the UV editor. If off only the selected faces are displayed in the UV editor. These two modes have very different results when transforming components in the UV editor.

Xem thêm

Selecting in the 3D Viewport.

Chế Độ lựa chọn

Select Modes dependent on the Sync Selection.

Tắt Đồng Bộ Hóa Chọn Lựa

Điểm Đỉnh

Select individual vertices.

Cạnh

Mở Rộng/Thu Nhỏ Chọn Lựa -- Select More/Less.

Bề Mặt

Lựa Chọn các Điểm Trùng Lặp.

Hải Đảo

Select contiguous groups of faces.

Bật Đồng Bộ Hóa Chọn Lựa

When selecting UVs or Edges, it behaves like the Shared Vertex option of the Sticky Selection Mode. When selecting Faces, it behaves like the Disabled option of the Sticky Selection Mode.

Điểm Đỉnh

Select individual vertices.

Cạnh

Mở Rộng/Thu Nhỏ Chọn Lựa -- Select More/Less.

Bề Mặt

Lựa Chọn các Điểm Trùng Lặp.

Chế Độ Lựa Chọn Dính Kết

This selector lets you enable automatic additional selection.

Có cùng điểm đỉnh

Selects UVs that share a mesh vertex, even if they are in different UV locations.

Có Cùng Vị Trí

Selects UVs that are in the same UV location and share a mesh vertex.

Tắt

Disables Sticky Selection. When you move a UV in this mode, each face owns its own UVs, allowing them to be separated.

Trình Đơn

Lựa Chọn A

Click and drag to box select UV coordinates. Alternatively, use B to start box selection.

Góc Độ Nhảy Cách Ctrl

Use the box lasso to select only pinned UV coordinates.

Lựa Chọn bằng Vòng Tròn

See Lựa Chọn bằng Vòng Tròn.

Chọn Toàn Bộ A

Tọa Độ UV.

Không/hủy chọn toàn bộ các Alt-A

Deselects all UV coordinate.

Đảo Nghịch Ctrl-I

Đảo nghịch lựa chọn Hiện Tại.

Thêm/Bớt Ctrl-Dấu Cộng (+) Bàn Số (NumpadPlus), Ctrl-Dấu Trừ (-) Bàn Số (NumpadMinus)

Expands/Contracts the selection to/from the adjacent elements of the selection type.

Chọn Cái Đã Ghim Lại Shift-P

Selects all pinned UVs.

Chọn Cái Đã Kết Nối
Kết Nối Ctrl-L

This operator selects all UVs that are connected to currently selected UVs. This works similarly to the tools in 3D Viewport.

Đường Ngắn Nhất

Path between two selected elements.

Phân Tách Lựa Chọn Y

Cuts apart the selected UVs from the map. Only those UVs which belong to fully selected faces remain selected. As the name implies, this is particularly useful to unlink faces and move them elsewhere. The hotkey is analogous to the mesh Split tool.

Chọn cái Chồng Gối Lên Nhau

Selects any UVs that are extended over other UVs while also selecting any underlying UVs.

Đường Ngắn Nhất

Tham Chiếu

Chế Độ

Chế Độ Biên Soạn

Trình Đơn

Chọn Lựa (Select) ‣ Nới Rộng/Thu Nhỏ (More/Less)

Tổ Hợp Phím Tắt

Hòa Trộn Ctrl-0

Selects all UV components along the shortest path from the active component to the one which was selected.

Bước Bậc Thang theo Bề Mặt

Supports diagonal paths for vertices and faces, and selects edge rings with edges.

Khoảng cách Cấu Trúc Liên Kết

Only takes into account the number of edges of the path and not the length of the edges to calculate the distances.

Phủ Kín Vùng Shift-Ctrl-NCT (LMB)

Selects all elements in the shortest paths from the active selection to the clicked area.

Các Tùy Chọn về Bỏ Chọn So Le

Allows to quickly select alternate elements in a path.

Số Phần Tử Không Chọn

The number of deselected elements in the repetitive sequence.

Cái được Chọn

The number of selected elements in the repetitive sequence.

Dịch Chuyển

Dịch chuyển từ điểm khởi đầu.

Mesh edit Select Shortest Path.

Chọn các Vòng Mạch

Tham Chiếu

Chế Độ

Chế Độ Biên Soạn

Tổ Hợp Phím Tắt

Alt-LMB, or Shift-Alt-LMB for modifying existing selection.

Holding Alt while selecting a UV component selects a loop of edges that are connected in a line end-to-end, passing through the edge under the mouse pointer. Holding Shift-Alt while clicking adds to the current selection.

Biên Soạn

Tạo Mới

Tham Chiếu

Chế Độ

Tổng Thể

Trình Đơn

Cốt (Armature) ‣ Phụ Huynh (Parent)

Tổ Hợp Phím Tắt

NumpadSlash

Creates a new Do Máy Sinh Tạo Image.

Mở

Tham Chiếu

Chế Độ

Tổng Thể

Trình Đơn

Cốt (Armature) ‣ Phụ Huynh (Parent)

Tổ Hợp Phím Tắt

NumpadSlash

Sao Chép Hình Ảnh từ Tầng Lớp.

Mở Bộ Nhớ Đệm Kết Xuất

Tham Chiếu

Chế Độ

Tổng Thể

Trình Đơn

Cốt (Armature) ‣ Phụ Huynh (Parent)

Tổ Hợp Phím Tắt

Ctrl-R

Load the current scene's render layers from disk cache, if available. This can be used to save RAM while rendering because the render layers do not have to be saved in RAM. And also for recovering some information from a fail render. For this to work, Cache Result must be enabled.

Thay Thế

Tham Chiếu

Chế Độ

Tổng Thể

Trình Đơn

Xương (Bone) ‣ Mối Quan Hệ (Relations)

Tổ Hợp Phím Tắt

Ctrl-R

Replaces the current image throughout the blend-file with another image.

Tái Nạp

Tham Chiếu

Chế Độ

Tổng Thể

Trình Đơn

Xương (Bone) ‣ Mối Quan Hệ (Relations)

Tổ Hợp Phím Tắt

Alt-R

Reload the image from the file on drive.

Biên Soạn Ngoại Tuyến

Tham Chiếu

Chế Độ

Tổng Thể

Trình Đơn

Thanh Bên (Sidebar) ‣ Thẻ Biên Soạn (Edit tab)

Using the Edit Externally tool Blender will open an external image editor, as specified in the Preferences and load in the image to be edited.

Lưu

Tham Chiếu

Chế Độ

Tổng Thể

Trình Đơn

Cốt (Armature) ‣ Phụ Huynh (Parent)

Tổ Hợp Phím Tắt

Alt-S

Save the image, if the image is already a file.

Quan trọng

Rendered images are not automatically saved, they have to be saved to drive manually.

Lưu Thành

Tham Chiếu

Chế Độ

Tổng Thể

Trình Đơn

Cốt (Armature) ‣ Phụ Huynh (Parent)

Tổ Hợp Phím Tắt

Thêm :kbd:`Shift-Ctrl- = `

Save the (rendered) image in a separate file.

Lưu một Bản Sao

Tham Chiếu

Chế Độ

Tổng Thể

Trình Đơn

Cốt (Armature) ‣ Bộ Xương (Skeleton)

Using Save as Copy will save the file to a specified name, but will keep the old one open in the Image editor.

Lưu Tất Cả các Hình Ảnh

Tham Chiếu

Chế Độ

Tổng Thể

Trình Đơn

Cốt (Armature) ‣ Bộ Xương (Skeleton)

Save all modified images. Packed images will be repacked.

Đảo Nghịch

Tham Chiếu

Chế Độ

Tổng Thể

Trình Đơn

Cốt (Armature) ‣ Phụ Huynh (Parent)

Đảo Nghịch Màu Hình Ảnh

Invert the colors of an image.

Đảo Nghịch Kênh

Đỏ, Lục, Lam -- Red, Green, Blue

Đổi Kích Thước

Tham Chiếu

Chế Độ

Tổng Thể

Trình Đơn

Xương (Bone) ‣ Mối Quan Hệ (Relations)

Adjust the image size in pixels.

Đảo Lật

Tham Chiếu

Chế Độ

Tổng Thể

Trình Đơn

Xương (Bone) ‣ Mối Quan Hệ (Relations)

This operator mirrors the image across a specified axis. Use this to alter the perspective of an image giving an alternative viewpoint; this can make an image more visually appealing or highlight some visual flaw.

Theo Chiều Ngang

Mirrors the image so the left side becomes the right side.

Theo Chiều Dọc

Mirrors the image so the top becomes the bottom.

Đóng Gói

Tham Chiếu

Chế Độ

Tổng Thể

Trình Đơn

Cốt (Armature) ‣ Phụ Huynh (Parent)

Packs the image into the blend-file. See Dữ liệu Đóng Gói.

Mở Gói

Tham Chiếu

Chế Độ

Tổng Thể

Trình Đơn

Cốt (Armature) ‣ Phụ Huynh (Parent)

Unpack the image to a drive.

Trích Xuất Bảng Pha Màu

Tham Chiếu

Chế Độ

Tổng Thể

Trình Đơn

Cốt (Armature) ‣ Phụ Huynh (Parent)

Extracts a Color Palette from the image for use by other tools.

Sinh Tạo Bút Chì Dầu

Tham Chiếu

Chế Độ

Tổng Thể

Trình Đơn

Cốt (Armature) ‣ Phụ Huynh (Parent)

Creates a Grease Pencil object using the currently selected image as a source.
